## 1. PromptCraft (프롬프트크래프트)

> AI 프롬프트를 공유·거래·평가하는 마켓플레이스 + 개인 프롬프트 관리 도구

| 항목 | 내용 |
|------|------|
| **타겟** | AI 도구를 일상적으로 사용하는 크리에이터, 마케터, 개발자 (25~45세) |
| **수익 모델** | 거래 수수료 15%, 프리미엄 구독 $9.99/월, 기업 팀 라이선스 |
| **기술 스택** | Next.js + Vercel, Supabase, OpenAI/Anthropic API, Stripe |

**핵심 기능:**
1. **프롬프트 라이브러리** — 카테고리별 프롬프트 탐색·저장, 버전 관리
2. **마켓플레이스** — 고품질 프롬프트 판매·구매, 리뷰 + 결과물 갤러리
3. **AI 프롬프트 옵티마이저** — AI가 프롬프트를 분석·개선, A/B 테스트

**차별화:** 기존 PromptBase 등은 단순 판매. "AI가 프롬프트를 개선해주는" 메타 도구 + 커뮤니티 결합. 한국어 특화 카테고리.

## 2. MeetingMind (미팅마인드)

> 회의를 자동 녹음·요약하고 액션 아이템을 추출해 PM 도구에 자동 연동하는 AI 비서

| 항목 | 내용 |
|------|------|
| **타겟** | 스타트업/중소기업 팀리더, PM, 리모트 워커 |
| **수익 모델** | 개인 $7/월, 팀 $15/인/월, 기업 커스텀, 연동 API 과금 |
| **기술 스택** | Flutter, Supabase, Whisper/Deepgram, GPT-4o, Zapier/n8n |

**핵심 기능:**
1. **실시간 회의 전사** — 한국어+영어 자동 인식, 화자 분리, 실시간 자막
2. **스마트 요약 & 액션 추출** — 3줄 요약 + 의사결정사항 + 할 일 목록 자동 정리
3. **워크플로우 자동화** — 액션 아이템을 Notion, Jira, Slack에 자동 생성

**차별화:** 한국어 회의 문화(존댓말, 간접표현) 이해 특화. "이거 다음에 한번 해보죠" = 액션 아이템으로 파싱.

## 3. AutoContent (오토콘텐츠)

> 블로그 글 하나로 인스타 캐러셀, 유튜브 스크립트, 뉴스레터, 트윗 스레드를 자동 생성

| 항목 | 내용 |
|------|------|
| **타겟** | 1인 크리에이터, 소규모 마케팅팀, 블로거, 인플루언서 |
| **수익 모델** | 크리에이터 $12/월, 비즈니스 $29/월, 종량제 |
| **기술 스택** | Next.js + Vercel, Supabase, Claude/GPT API, Buffer/Later API |

**핵심 기능:**
1. **원클릭 멀티포맷 변환** — 하나의 원본 → 10개+ 플랫폼 최적화 포맷
2. **브랜드 톤 학습** — 기존 콘텐츠 학습으로 일관된 톤 유지
3. **퍼블리싱 스케줄러** — 각 플랫폼 최적 시간에 자동 게시 + 성과 대시보드

**차별화:** 텍스트 기반 멀티채널 변환 집중. 네이버 블로그, 카카오뷰, 브런치 포맷 지원.

## 4. MailPilot (메일파일럿)

> 이메일을 AI가 분류·요약·답장 초안까지 작성하는 스마트 이메일 어시스턴트

| 항목 | 내용 |
|------|------|
| **타겟** | 하루 50+통 이메일 비즈니스 전문가, 프리랜서, 영업직 |
| **수익 모델** | 프리미엄 $9.99/월, 팀 $19.99/인/월 |
| **기술 스택** | Next.js, Supabase, Gmail/Outlook API, GPT-4o |

**핵심 기능:**
1. **스마트 분류 & 우선순위** — 긴급/중요/정보성/스팸 자동 분류
2. **AI 요약 & 원클릭 답장** — 스레드 3줄 요약, 답장 초안 3가지 제안
3. **팔로업 자동화** — 조건부 자동 리마인드 설정

**차별화:** 합리적 가격 + 한국어 비즈니스 이메일(인사말, 존칭) 특화. Chrome 확장 제공.

## 5. SkillClone (스킬클론)

> 개인의 전문 지식을 학습해 "나 대신 답변하는" AI 분신을 만드는 플랫폼

| 항목 | 내용 |
|------|------|
| **타겟** | 컨설턴트, 코치, 전문가, 인플루언서 |
| **수익 모델** | SaaS 구독 $19~49/월, 상담 중개 10%, API 과금 |
| **기술 스택** | Next.js + Vercel, Supabase, RAG + pgvector, Stripe |

**핵심 기능:**
1. **지식 업로드 & 학습** — 문서, Q&A, 강의 자료 → AI가 지식 체계 학습
2. **AI 분신 챗봇** — "나처럼" 답변하는 챗봇, 웹사이트 임베드 가능
3. **수익화 연동** — AI 답변 → 부족하면 실제 전문가 연결, Stripe 결제

**차별화:** "전문가 경제" 특화. 크몽, 탈잉 유저 타겟.

## 6. DataTalk (데이터톡)

> 엑셀/CSV/DB를 업로드하면 자연어로 질문해서 차트와 인사이트를 얻는 노코드 데이터 분석

| 항목 | 내용 |
|------|------|
| **타겟** | SQL/Python 모르는 마케터, 사업 기획자, 소상공인 |
| **수익 모델** | 프리미엄 $14.99/월, 팀 $29/인/월, 기업 온프레미스 |
| **기술 스택** | Next.js, Supabase, DuckDB, GPT-4o, D3.js/Recharts |

**핵심 기능:**
1. **자연어 쿼리** — "지난달 가장 많이 팔린 상품은?" → SQL/차트 자동 변환
2. **자동 인사이트** — 이상치, 트렌드, 상관관계 자동 발견 → 한국어 리포트
3. **대시보드 빌더** — 드래그앤드롭 대시보드, 자동 갱신 + 팀 공유

**차별화:** 브라우저 내 분석(DuckDB)으로 데이터가 서버를 거치지 않는 프라이버시 우선 설계.

## 7. VoiceForge (보이스포지)

> 내 목소리를 클로닝해 다국어 더빙, 오디오북, 팟캐스트 나레이션을 자동 생성

| 항목 | 내용 |
|------|------|
| **타겟** | 유튜버, 팟캐스터, 온라인 강사, 오디오 콘텐츠 크리에이터 |
| **수익 모델** | 크레딧제 $9.99/60분, 프로 $24.99/월, 상업적 라이선스 별도 |
| **기술 스택** | Next.js, Supabase, ElevenLabs/XTTS API, FFmpeg |

**핵심 기능:**
1. **음성 클로닝** — 3분 녹음으로 AI 모델 생성, 텍스트 → 내 목소리 변환
2. **다국어 더빙** — 내 목소리로 영어/일어/중국어 더빙 자동 생성 (립싱크 포함)
3. **편집 스튜디오** — 텍스트 수정 = 오디오 수정, BGM 자동 매칭

**차별화:** "녹음→편집→배포" 원스톱 크리에이터 스튜디오. 한국어 음성 품질 특화.

## 8. CodeBuddy (코드버디)

> 코딩 초보자를 위한 AI 페어프로그래밍 파트너 — 답을 주지 않고 힌트부터

| 항목 | 내용 |
|------|------|
| **타겟** | 비전공자, 전직 희망자, 부트캠프 수강생 |
| **수익 모델** | 프리미엄 $7.99/월, 부트캠프 B2B, 취업 연계 |
| **기술 스택** | Next.js, Supabase, Monaco Editor, Claude API, GitHub API |

**핵심 기능:**
1. **대화형 코딩 튜터** — 에러 설명 + 수정 방법, 소크라테스식 힌트 방식
2. **프로젝트 기반 학습** — "TODO 앱 만들기" 같은 실전 프로젝트 단계별 안내
3. **포트폴리오 빌더** — 학습 프로젝트 → GitHub 자동 커밋 + 포트폴리오 페이지

**차별화:** "배우는 사람" 위한 설계. 소크라테스식 질문으로 사고력 향상. 한국어 교육 콘텐츠.

## 10. PlannerAI (플래너AI)

> 일정·할일·습관·목표를 통합 관리하며 AI가 최적 일정과 루틴을 자동 설계

| 항목 | 내용 |
|------|------|
| **타겟** | 직장인, 대학생, 프리랜서 — 시간 관리 관심 20~40대 |
| **수익 모델** | 프리미엄 $5.99/월, 연간 $49.99, 기업 웰빙 B2B |
| **기술 스택** | Flutter, Supabase, Google Calendar API, GPT-4o-mini, FCM |

**핵심 기능:**
1. **AI 일정 최적화** — Google Calendar 연동, 할 일 → 최적 시간대 자동 블록 배치
2. **습관 트래커 + 넛지** — 목표 기반 습관 설정, AI가 최적 타이밍에 동기부여
3. **주간/월간 회고 리포트** — 달성률, 시간 사용 패턴, AI 맞춤 피드백

**차별화:** "일정+할일+습관+목표" 통합 + AI 라이프 타임라인 최적화. "갓생살기" 트렌드.
