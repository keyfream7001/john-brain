## 1. SleepLab (슬립랩)

> 수면 데이터를 AI로 분석해 개인 맞춤 수면 개선 프로그램을 제공하는 수면 코칭 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 수면 문제를 겪는 20~50대 직장인, 불면증 경향 현대인 |
| **수익 모델** | 프리미엄 $6.99/월, 기업 복지 B2B, 수면용품 제휴 |
| **기술 스택** | Flutter, Supabase, HealthKit/Health Connect, TensorFlow Lite, GPT-4o-mini |

**핵심 기능:**
1. **스마트 수면 트래킹** — 스마트워치 연동 수면 단계(REM, 깊은/얕은 수면) 분석
2. **AI 수면 코치** — 7일 데이터 → 맞춤 수면 루틴(취침시간, 카페인, 조명) 제안
3. **사운드스케이프 & CBT-I** — 백색소음/자연음 + 인지행동치료 기반 수면 프로그램

**차별화:** "분석 → 맞춤 처방 → 행동 변화" 코칭. CBT-I 디지털화로 수면의학 근거 기반.

## 2. MealMate (밀메이트)

> 음식 사진을 찍으면 AI가 영양소를 분석하고 건강 목표에 맞는 다음 식사를 추천

| 항목 | 내용 |
|------|------|
| **타겟** | 다이어트 중인 20~40대, 헬스 애호가, 만성질환 관리자 |
| **수익 모델** | 프리미엄 $5.99/월, 영양 보충제 제휴, 병원/헬스장 B2B |
| **기술 스택** | Flutter, Supabase, Vision API + 커스텀 모델, GPT-4o |

**핵심 기능:**
1. **AI 사진 인식 식단 기록** — 사진 한 장 → 음식명, 칼로리, 탄단지 자동. 한국 음식 DB 특화
2. **맞춤 식단 플래너** — 목표별 주간 식단 자동 생성. 배달앱 메뉴도 분석
3. **영양소 트렌드 대시보드** — 주간/월간 영양소 추이, 부족 영양소 경고

**차별화:** 한국 식약처 영양 DB + 한국 음식 특화 인식모델. "배달시킬 건데 뭐 먹지?" 기능.

## 3. MindSpace (마인드스페이스)

> AI 기반 인지행동치료(CBT) + 명상 + 무드 트래킹 디지털 멘탈헬스 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 불안·우울감 20~30대, 스트레스 관리 직장인, 상담 접근 어려운 사람 |
| **수익 모델** | 프리미엄 $8.99/월, 기업 EAP B2B, 상담사 연결 수수료 |
| **기술 스택** | Flutter, Supabase (암호화), GPT-4o, HealthKit |

**핵심 기능:**
1. **AI 감정 저널** — 매일 감정 기록 → 패턴 분석 → 인사이트 제공
2. **CBT 워크북** — 디지털 인지행동치료 프로그램, AI 치료사 대화형
3. **SOS 모드 & 전문가 연결** — 급성 스트레스 시 호흡법/그라운딩 + 상담사 연결

**차별화:** 임상 근거 기반 CBT를 AI로 접근 가능하게. 한국어 상담 문화 + 정신건강복지센터 연계.

## 4. WalkQuest (워크퀘스트)

> 걷기를 RPG 게임으로 만드는 게이미피케이션 걷기 앱 — 걸을수록 캐릭터 성장

| 항목 | 내용 |
|------|------|
| **타겟** | 운동 습관 없는 20~40대, 게임 좋아하는 MZ세대 |
| **수익 모델** | 인앱 구매(코스메틱, 스토리팩), 프리미엄 $4.99/월, 지자체/건보 B2G |
| **기술 스택** | Flutter, Supabase, ARCore/ARKit, 만보계 센서 API, Firebase |

**핵심 기능:**
1. **스토리 기반 걷기** — 판타지/미스터리 스토리를 걸으면서 진행, 걸음수로 챕터 해금
2. **캐릭터 성장 & 아이템** — 걸음수 = 경험치, 레벨업, 아이템, 길드 시스템
3. **AR 탐험 모드** — 동네 산책길에 AR 오브젝트 배치, 걷기 건강 집중

**차별화:** "걷기 = 게임 플레이"로 습관 형성. 한국 동네 산책로, 둘레길 로컬 퀘스트.

## 5. PillPal (필팔)

> 복약 알림 + 약물 상호작용 검사 + 처방전 관리를 하나로

| 항목 | 내용 |
|------|------|
| **타겟** | 만성질환자, 다약제 복용 노인, 환자 보호자, 건강기능식품 복용자 |
| **수익 모델** | 프리미엄 $3.99/월, 제약사 광고, 약국 배달 제휴 |
| **기술 스택** | Flutter, Supabase, OCR, KIMS/약학정보원 API, FCM |

**핵심 기능:**
1. **스마트 복약 알림** — 처방전 사진 스캔 → 자동 스케줄 생성, 복용 확인/미복용 추적
2. **약물 상호작용 경고** — 약+영양제 간 위험한 상호작용 실시간 경고
3. **가족 관리 모드** — 부모님 복약 상태 원격 확인, 미복용 시 보호자 알림

**차별화:** 상호작용 검사 + 가족 돌봄. 한국 건강보험 처방전 시스템 연동 가능.

## 6. StretchFlow (스트레치플로우)

> 직장인 거북목·어깨결림을 위한 AI 자세 분석 + 맞춤 스트레칭 가이드

| 항목 | 내용 |
|------|------|
| **타겟** | 장시간 데스크워크 직장인, 재택근무자, 자세 교정 필요 20~40대 |
| **수익 모델** | 프리미엄 $4.99/월, 기업 복지 B2B, 인체공학 제품 제휴 |
| **기술 스택** | Flutter, MediaPipe/MoveNet, Supabase |

**핵심 기능:**
1. **AI 자세 분석** — 카메라로 앉은 자세 분석, 거북목/라운드숄더/골반 틀어짐 진단
2. **맞춤 스트레칭 루틴** — 진단 기반 3~10분 스트레칭 영상 + 타이머 + 음성 안내
3. **업무 중 리마인더** — 포모도로 + 5분 스트레칭 알림 결합

**차별화:** "업무 중 자세 관리" 특화. 한국 직장인 회사 환경에 맞는 짧은 스트레칭.

## 7. CycleWise (사이클와이즈)

> 여성 건강 주기 추적 + AI 기반 증상 예측 + 호르몬 기반 라이프스타일 추천

| 항목 | 내용 |
|------|------|
| **타겟** | 생리주기 관리 필요 15~45세 여성, 임신 계획 커플 |
| **수익 모델** | 프리미엄 $5.99/월, 여성 건강용품 제휴, 산부인과 예약 연계 |
| **기술 스택** | Flutter, Supabase (암호화), TensorFlow Lite, HealthKit |

**핵심 기능:**
1. **AI 주기 예측** — 개인 패턴 학습, 불규칙 주기도 점진적 정확도 향상
2. **증상 예측 & 관리** — "3일 후 PMS 두통 가능성" 사전 알림 + 대처법
3. **호르몬 싱크 플래너** — 주기 단계별 최적 운동·식단·업무 스타일 추천

**차별화:** "주기에 따라 생활을 최적화"하는 라이프스타일 플래너. 한국 산부인과/약국 연계.

## 8. ElderCare (엘더케어)

> 독거노인/고령 부모의 건강·안전을 원격으로 모니터링하고 돌보는 가족 케어 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 부모님과 떨어져 사는 30~50대, 독거노인, 요양보호사 |
| **수익 모델** | 가족 구독 $9.99/월, 요양기관 B2B, IoT 디바이스 번들 |
| **기술 스택** | Flutter, Supabase, HealthKit/Health Connect, Twilio, FCM |

**핵심 기능:**
1. **활동 모니터링** — 스마트워치 기반 걸음수, 수면, 심박수. 24h 비활동 시 경고
2. **낙상 감지 & SOS** — 웨어러블 센서 낙상 감지, 30초 미응답 시 119 자동 신고
3. **건강 일지 공유** — 혈압, 혈당, 복약 기록 가족 앱에서 실시간 확인

**차별화:** "건강 데이터 기반 종합 돌봄" + "낙상 자동 감지". 한국 고령화 + 효문화 맞춤.

## 9. GutCheck (거체크)

> 장 건강과 소화 상태를 매일 기록·분석하고 장내 미생물 기반 맞춤 식이요법 추천

| 항목 | 내용 |
|------|------|
| **타겟** | IBS 환자, 소화 불량 잦은 직장인, 장 건강 관심 건강족 |
| **수익 모델** | 프리미엄 $6.99/월, 마이크로바이옴 검사 키트 제휴, 건기식 커머스 |
| **기술 스택** | Flutter, Supabase, GPT-4o-mini, 자체 음식-증상 상관관계 모델 |

**핵심 기능:**
1. **증상 일지** — 소화 상태, 브리스톨 척도, 복부 증상 간편 기록 + 음식 상관관계 분석
2. **트리거 음식 탐지** — "유제품이 트리거일 가능성 78%" 같은 AI 분석
3. **마이크로바이옴 연동** — 장내 미생물 검사 결과 연동 시 맞춤 프로바이오틱스 추천

**차별화:** "먹은 음식 → 장 반응" 인과관계 추적. 한국인 매운 음식 문화 최적화.

## 10. BreathEasy (브리스이지)

> 실시간 공기질 + 알레르기 지수 모니터링 + 개인 호흡기 건강 관리 통합 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 알레르기성 비염/천식 환자, 미세먼지 민감자, 아이 있는 부모 |
| **수익 모델** | 프리미엄 $3.99/월, IoT 공기질 센서 판매, 공기청정기 제휴 |
| **기술 스택** | Flutter, Supabase, 한국환경공단 API + AirVisual, TensorFlow |

**핵심 기능:**
1. **초정밀 공기질 맵** — 공공 + IoT 센서 결합 블록 단위 공기질 지도 (미세먼지·오존·꽃가루)
2. **개인 알레르기 프로필** — 트리거 설정 → 위험 수준 사전 알림
3. **외출 플래너** — AI 시간대별 공기질 예측 → 최적 외출 시간 추천

**차별화:** "내 알레르기 프로필 × 공기질 = 맞춤 행동 추천". 한국 봄 미세먼지·꽃가루 최적.
