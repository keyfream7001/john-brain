## 1. SplitWise+ (스플릿와이즈플러스)

> 더치페이 자동 정산 + 그룹 예산 관리 + 공동 저축 목표를 하나로

| 항목 | 내용 |
|------|------|
| **타겟** | 정기 모임 20~30대, 동거 커플, 룸메이트, 소모임 |
| **수익 모델** | 프리미엄 $2.99/월, 송금 수수료 0.5%, 금융상품 제휴 |
| **기술 스택** | Flutter, Supabase, OCR, 토스/카카오페이 API, GPT-4o-mini |

**핵심 기능:**
1. **AI 자동 정산** — 영수증 사진 → 항목별 인식 → 1/N or 개별 정산 + 송금 링크
2. **그룹 예산 대시보드** — 월 모임비, 여행 경비 실시간 추적
3. **공동 저축 목표** — "제주도 여행 100만원 모으기" 공동 목표 + 진행률 시각화

**차별화:** "정산+예산+저축" 그룹 재무관리 통합. 한국 송금 시스템 완벽 연동.

## 2. MicroVest (마이크로베스트)

> 잔돈 자동 투자 + AI 포트폴리오 추천으로 MZ세대 첫 투자를 돕는 소액 투자 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 투자 시작 엄두 안 나는 사회 초년생, 대학생, 20대 |
| **수익 모델** | 운용보수 AUM 0.5%, 프리미엄 $3.99/월, 제휴 금융상품 |
| **기술 스택** | Flutter, Supabase, 증권사 API, GPT-4o, Plaid |

**핵심 기능:**
1. **잔돈 라운드업 투자** — 3,720원 결제 → 280원 자동 투자
2. **AI 포트폴리오 추천** — 성향 퀴즈 → ETF/주식 자동 구성 및 리밸런싱
3. **투자 교육 콘텐츠** — 맥락형 교육 + 투자 용어 사전 + 퀴즈

**차별화:** "잔돈이 알아서 투자되는" 완전 자동화. Acorns 한국판 + 교육 + AI 분석.

## 3. TokenHome (토큰홈)

> 부동산 조각투자 — 10만원부터 부동산에 투자하고 월 임대수익 배당

| 항목 | 내용 |
|------|------|
| **타겟** | 목돈 없는 20~30대 부동산 관심자, 소액 분산 투자자 |
| **수익 모델** | 거래 수수료 1%, 운용보수 연 0.8%, 토큰 발행 수수료 |
| **기술 스택** | Next.js + Vercel, Supabase, Solana/Polygon, KYC/AML API |

**핵심 기능:**
1. **부동산 토큰 마켓** — 오피스텔/상가/물류센터 토큰화, 10만원 단위 구매
2. **월 배당금 자동 지급** — 지분 비율 자동 계산·분배, 재투자 옵션
3. **AI 투자 분석** — 지역별 공실률, 시세, 인구 트렌드 기반 투자 등급

**차별화:** 블록체인 기반 토큰 유동성(2차 거래) 강화 + AI 분석.

## 4. SpendSense (스펜드센스)

> 소비 패턴 AI 분석 → "돈 새는 곳" 진단 + 맞춤 절약 전략

| 항목 | 내용 |
|------|------|
| **타겟** | 돈 어디로 사라지는지 모르는 20~30대, 저축률 높이고 싶은 직장인 |
| **수익 모델** | 프리미엄 $4.99/월, 금융상품 추천 리퍼럴, 제휴 할인 쿠폰 |
| **기술 스택** | Flutter, Supabase, 오픈뱅킹 API, GPT-4o-mini |

**핵심 기능:**
1. **자동 분류 가계부** — 은행/카드 자동 연동 + AI 카테고리 자동 분류
2. **이상 지출 알림** — "이번 달 카페 지출 40% 높아요" 실시간 경고
3. **절약 챌린지** — AI 분석 기반 맞춤 챌린지 제안, 달성 시 뱃지·리워드

**차별화:** "분석 → 문제 진단 → 행동 변화 유도". 게이미피케이션 절약 챌린지.

## 5. FreelancePay (프리랜스페이)

> 프리랜서 올인원 재무 관리 — 견적·인보이스·수금·세금 신고 자동화

| 항목 | 내용 |
|------|------|
| **타겟** | 프리랜서, 1인 사업자, 크리에이터 |
| **수익 모델** | 프리미엄 $9.99/월, 인보이스당 0.5%, 세무사 연결 수수료 |
| **기술 스택** | Next.js, Flutter, Supabase, 홈택스/국세청 API, GPT-4o |

**핵심 기능:**
1. **AI 견적·인보이스 생성** — 프로젝트 설명 → 전문적 견적서/인보이스 자동 생성
2. **수금 자동화** — 미수금 알림, 자동 리마인더, 입금 확인 자동 처리
3. **세금 대시보드** — 소득·경비 자동 정리, 종합소득세 예상, 홈택스 연동

**차별화:** 한국 3.3% 원천징수, 종합소득세, 부가세 신고 완벽 지원.

## 6. CryptoSimple (크립토심플)

> 암호화폐 초보자 위한 자동 DCA + AI 시장 해설 + 포트폴리오 관리

| 항목 | 내용 |
|------|------|
| **타겟** | 비트코인 관심있지만 시작 못한 2030, 장기 투자 성향 초보자 |
| **수익 모델** | 매수 수수료 0.5%, 프리미엄 $5.99/월, 거래소 제휴 |
| **기술 스택** | Flutter, Supabase, 업비트/빗썸 API, GPT-4o, CoinGecko API |

**핵심 기능:**
1. **원클릭 DCA 설정** — 매주/매월 자동 분할 매수
2. **AI 시장 뉴스 해설** — "FED 금리 동결 → 비트코인 영향" 초보자 눈높이
3. **리스크 대시보드** — 포트폴리오, 수익률, 공포탐욕지수 시각화

**차별화:** "설정하고 잊어라" DCA + 교육으로 장기 투자자 온보딩 특화.

## 7. SubTracker (서브트래커)

> 내가 구독 중인 모든 서비스 한눈에 — 관리 + 해지 알림 + 절약 제안

| 항목 | 내용 |
|------|------|
| **타겟** | 구독 5개+ 사용하는 모든 사람, 구독 지출 파악 못하는 2030 |
| **수익 모델** | 프리미엄 $2.99/월, 대안 서비스 추천 리퍼럴, 제휴 할인 |
| **기술 스택** | Flutter, Supabase, 오픈뱅킹 API, GPT-4o-mini |

**핵심 기능:**
1. **자동 구독 탐지** — 카드 내역에서 정기 결제 자동 인식·분류
2. **무료체험 만료 알림** — 불필요한 과금 방지 사전 알림
3. **절약 추천** — "넷플릭스 광고형으로 바꾸면 월 5,500원 절약"

**차별화:** 한국 카드사 연동 + 국내 구독(멜론, 왓챠, 쿠팡 등) 완벽 지원.

## 8. TipJar (팁자)

> 크리에이터 위한 간편 후원/팁 플랫폼 — 링크 하나로 글로벌 후원, 수수료 최소화

| 항목 | 내용 |
|------|------|
| **타겟** | 유튜버, 인디 뮤지션, 웹소설 작가, 팟캐스터, 오픈소스 개발자 |
| **수익 모델** | 후원 수수료 3%(업계 최저), 프리미엄 $7.99/월 |
| **기술 스택** | Next.js + Vercel, Supabase, Stripe/토스, Circle API (USDC) |

**핵심 기능:**
1. **원링크 후원 페이지** — 3분 만에 후원 페이지 생성 (일회/월정기/목표)
2. **글로벌 결제** — 카카오페이, 토스, PayPal, USDC 동시 지원
3. **팬 관리 CRM** — 후원자 목록, 랭킹, 자동 감사 메시지, 전용 콘텐츠

**차별화:** 3% 최저 수수료 + 한국+글로벌 결제 동시 지원.

| 항목 | 내용 |
|------|------|

| **수익 모델** | 프리미엄 $14.99/월, 세무사/회계사 연결, 사업자 대출 리퍼럴 |
| **기술 스택** | Next.js, Supabase, 오픈뱅킹/세금계산서 API, GPT-4o |

**핵심 기능:**
1. **자동 매출·비용 트래킹** — 사업자 통장/카드 연동, 세금계산서 OCR

**차별화:** "사업자 재무 전체 그림" + AI 경영 의사결정 지원.

## 10. PayBridge (페이브릿지)

> 해외 프리랜서/디지털 노마드 위한 크로스보더 결제 + 환전 + 세금 원스톱

| 항목 | 내용 |
|------|------|
| **타겟** | 해외 클라이언트 한국 프리랜서, 해외 거주 한국인, 디지털 노마드 |
| **수익 모델** | 환전 스프레드 0.3%, 송금 수수료, 프리미엄 $12.99/월 |
| **기술 스택** | Next.js + Vercel, Supabase, Wise/Stripe API, GPT-4o |

**핵심 기능:**
1. **글로벌 인보이스 & 수금** — USD/EUR/JPY 인보이스, 자동 원화 환전
2. **스마트 환전** — AI 환율 추이 분석 → 최적 환전 타이밍 추천
3. **이중 세금 관리** — 해외 소득 신고, 외국납부세액공제, 종합소득세 통합

**차별화:** "해외 수금 → 환전 → 세금 신고" 원스톱. 디지털 노마드 급증 트렌드.
