## 1. CircleUp (써클업)

> 관심사 기반 소규모 그룹(5~15명) 자동 매칭 + 정기 오프라인 모임 연결

| 항목 | 내용 |
|------|------|
| **타겟** | 새 사람 만나고 싶지만 대규모 부담인 25~35세 직장인 |
| **수익 모델** | 프리미엄 $5.99/월, 모임 장소 제휴, 이벤트 티켓 수수료 |
| **기술 스택** | Flutter, Supabase, GPT-4o-mini, Google Maps API |

**핵심 기능:**
1. **AI 그룹 매칭** — 관심사 + 위치 + 성향 기반 소그룹 자동 구성
2. **아이스브레이킹 봇** — AI 대화 주제 제안, "이번 주 질문" 카드, 미션
3. **오프라인 모임 플래너** — 일정 투표, 장소 추천, 참석 확인 + 노쇼 방지

**차별화:** AI가 자동으로 그룹 구성+운영까지. 인트로버트 친화적.

## 2. TimeVault (타임볼트)

> 미래의 나에게 편지/사진/영상을 보내는 디지털 타임캡슐 소셜 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 추억을 소중히 여기는 20~30대, 연인·친구와 특별한 경험 |
| **수익 모델** | 프리미엄 $3.99/월, 특별 테마 캡슐, 기업 이벤트 |
| **기술 스택** | Flutter, Supabase, S3, Cloud Functions, FCM |

**핵심 기능:**
1. **타임캡슐 생성** — 텍스트/사진/영상/음성 → 1개월~10년 후 자동 오픈
2. **그룹 캡슐** — 졸업/결혼/출산 등 이벤트에 공동 메시지 투입
3. **추억 알림** — "1년 전 오늘 당신이 보낸 캡슐" 회고 기능

**차별화:** "미래에 열리는 감정" 설계. 노스탤지어 + 미래 기대감. 한국 기념일 문화.

## 3. HonestBox (어니스트박스)

> 완전 익명 솔직한 피드백과 고민 상담 — AI 독성 필터로 안전한 공간

| 항목 | 내용 |
|------|------|
| **타겟** | 솔직한 의견 듣고 싶은 20~30대, 직장 피드백 필요 팀 |
| **수익 모델** | 프리미엄 $3.99/월, 기업 익명 피드백 B2B $29/팀/월 |
| **기술 스택** | Next.js + Vercel, Supabase, GPT-4o, Perspective API |

**핵심 기능:**
1. **익명 피드백 보드** — "고쳐야 할 점?", "사업 아이디어 어때?" 익명 답변 수집
2. **AI 독성 필터** — 악성댓글, 인신공격 실시간 필터링. 건설적 피드백만 통과
3. **커뮤니티 고민 상담** — 연애/직장/진로 고민 익명 올리기 + AI 상담 보조

**차별화:** "AI 독성 필터"로 건설적 소통만. 한국 "눈치 문화"에서 솔직한 소통 니즈.

## 4. PetConnect (펫커넥트)

> 반려동물 보호자 소셜 네트워크 — 산책 메이트, 펫시터 매칭, 건강 커뮤니티

| 항목 | 내용 |
|------|------|
| **타겟** | 반려동물 보호자 1,500만+, 강아지·고양이 보호자 25~45세 |
| **수익 모델** | 펫시터 중개 15%, 동물병원/펫샵 광고, 용품 커머스 제휴 |
| **기술 스택** | Flutter, Supabase, Google Maps API, Stripe/토스 |

**핵심 기능:**
1. **동네 펫 소셜** — 산책 메이트, 동네 맛집/동물병원 리뷰
2. **펫시터 매칭** — 검증된 동네 펫시터, 실시간 사진/영상 리포트 + 보험
3. **건강 관리 허브** — 예방접종 스케줄, 병원 기록, 수의사 간편 상담

**차별화:** "소셜+돌봄+건강" 통합. 한국 반려동물 시장 30조원+.

## 5. Duolog (듀올로그)

> AI 매칭 외국인과 1:1 언어 교환 — 30분 반반 대화로 무료 어학 연습

| 항목 | 내용 |
|------|------|
| **타겟** | 영어/일어/중국어 실전 연습 한국인, 한국어 배우려는 외국인 |
| **수익 모델** | 프리미엄 $7.99/월, 어학원/유학원 광고, 교재 제휴 |
| **기술 스택** | Flutter, Supabase, WebRTC, Whisper, GPT-4o |

**핵심 기능:**
1. **AI 매칭** — 언어 레벨 + 관심사 + 시간대 기반, 노쇼 방지 신뢰도 시스템
2. **구조화된 대화 세션** — AI 주제 제안, 타이머 언어 전환, 실시간 번역
3. **AI 리뷰 & 피드백** — 녹음 분석 → 발음, 문법, 어휘 맞춤 피드백

**차별화:** "구조화된 세션 + AI 피드백"으로 학습 효과 극대화.

## 6. NeighborNote (네이버노트)

> 동네 소식·정보 실시간 공유 하이퍼로컬 커뮤니티 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 동네 정보 필요 주민 (맛집, 공사 소음, 분실물, 이벤트 등) |
| **수익 모델** | 소상공인 반경 1km 광고, 프리미엄 $1.99/월, 이벤트 티켓 수수료 |
| **기술 스택** | Flutter, Supabase, PostGIS, FCM |

**핵심 기능:**
1. **동네 피드** — 반경 1~3km 실시간 소식 (맛집/재난/분실물/이벤트/소음)
2. **이웃 투표** — 동네 이슈에 투표 + 토론
3. **로컬 이벤트 보드** — 플리마켓, 문화센터, 체육관 하이퍼로컬 이벤트

**차별화:** 순수 커뮤니티에 집중 + 투표·이벤트 "동네 시민 참여" 기능.

## 7. PhotoBattle (포토배틀)

> 매일 사진 주제로 촬영해 투표 대결하는 사진 게임 소셜 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 사진 즐기는 20~30대, 인스타와 다른 재미 찾는 MZ세대 |
| **수익 모델** | 인앱 구매(투표 부스트, 프레임), 프리미엄 $3.99/월, 카메라 제휴 |
| **기술 스택** | Flutter, Supabase, S3, ELO 레이팅 시스템 |

**핵심 기능:**
1. **데일리 포토 챌린지** — 매일 주제 발표("오늘의 그림자"), 24시간 내 제출
2. **블라인드 투표** — 익명 2개씩 보여주고 투표, 편견 없는 사진 경쟁
3. **리그 & 시즌** — 주간/월간 랭킹, 실력별 리그, 시즌별 테마

**차별화:** "촬영 실력 경쟁" 게임 메카닉 × 소셜. 창작 동기부여 + 매일 참여.

## 8. VoiceSpace (보이스스페이스)

> 음성만으로 소통하는 비동기 소셜 앱 — 텍스트 대신 짧은 음성 메시지

| 항목 | 내용 |
|------|------|
| **타겟** | 텍스트 피로 느끼는 20~40대, 장거리 연인/가족, 팟캐스트 청취층 |
| **수익 모델** | 프리미엄 $4.99/월, 음성 기반 광고, 크리에이터 후원 |
| **기술 스택** | Flutter, Supabase, S3 + CloudFront, Whisper, FFmpeg |

**핵심 기능:**
1. **음성 피드** — 30초~3분 음성 메시지 공유 (하루 일기, 추천 음악, 생각)
2. **음성 DM** — "워키토키" 스타일 비동기 음성 교환
3. **음성 커뮤니티** — 관심사별 음성 게시판, AI 자동 전사+요약

**차별화:** "비동기 음성"에 집중. 텍스트보다 따뜻하고, 영상보다 부담없는 중간 지점.

## 9. GratitudeJar (감사항아리)

> 매일 감사한 일 3가지 기록 + 친구와 공유하는 긍정 심리학 기반 웰빙 소셜

| 항목 | 내용 |
|------|------|
| **타겟** | 멘탈 웰빙 관심 20~40대, 번아웃 직장인, 긍정 습관 만들기 |
| **수익 모델** | 프리미엄 $2.99/월, 웰빙 콘텐츠 구독, 감사 굿즈 커머스 |
| **기술 스택** | Flutter, Supabase, GPT-4o-mini |

**핵심 기능:**
1. **데일리 감사 기록** — 1~3개 간단 기록, 사진 첨부, 연속 스트릭
2. **감사 서클** — 친구 3~5명과 공유, 하트·응원 리액션
3. **월간 회고 & 인사이트** — AI 분석 "가장 감사하는 카테고리는 '사람'"

**차별화:** "감사를 나누는 소셜". 긍정 심리학 연구 근거 + 건강 효과 마케팅.

## 10. MentorLink (멘토링크)

> 업계 선배와 후배 1:1 매칭 커리어 멘토링 프로페셔널 네트워킹 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 커리어 조언 필요 주니어(2~7년차), 멘토링 원하는 시니어 |
| **수익 모델** | 멘티 $14.99/월, 기업 HR B2B, 프리미엄 멘토 세션 수수료 |
| **기술 스택** | Next.js, Flutter, Supabase, WebRTC/Zoom API |

**핵심 기능:**
1. **AI 멘토 매칭** — 업종, 직무, 관심 분야 분석 → 최적 매칭
2. **구조화된 멘토링 프로그램** — 4~8주 프로그램 템플릿, 주차별 주제
3. **멘토링 세션 관리** — 미팅 예약, 세션 기록, 액션 아이템 추적

**차별화:** "지속적 멘토링 관계" 구조화된 프로그램 제공. 커피챗과 다른 깊이.
