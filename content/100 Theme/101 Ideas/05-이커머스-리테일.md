## 1. StyleMirror (스타일미러)

> AI + AR로 옷을 입어보지 않고 가상 피팅 — 체형 분석 + 스타일 추천

| 항목 | 내용 |
|------|------|
| **타겟** | 온라인 의류 쇼핑 사이즈 걱정 20~30대, 반품률 줄이려는 브랜드 |
| **수익 모델** | 쇼핑몰 제휴 CPA 5~10%, B2B 가상 피팅 솔루션, 프리미엄 $5.99/월 |
| **기술 스택** | Flutter, ARCore/ARKit, Supabase, Three.js, AI 체형 분석 |

**핵심 기능:**
1. **3D 바디 스캔** — 카메라 촬영 → 체형 데이터 추출 → 3D 아바타 생성
2. **AR 가상 피팅** — 관심 옷을 아바타에 입혀보기, 핏감·색상·코디 확인
3. **AI 스타일 추천** — 체형, 선호 스타일, 구매 이력 기반 개인 맞춤 큐레이션

**차별화:** 한국 패션 브랜드 + 한국인 체형 최적화. 쿠팡/무신사 연동.

## 2. DropDeal (드롭딜)

> 매일 정해진 시간에 한정 수량 특가 상품 "드롭" — 게임처럼 쇼핑

| 항목 | 내용 |
|------|------|
| **타겟** | 할인 민감 20~30대, 한정판/특가 좋아하는 MZ세대 |
| **수익 모델** | 판매 수수료 10~15%, 드롭 슬롯 광고비, 럭키박스 마진 |
| **기술 스택** | Flutter, Supabase, Redis, Stripe/토스 |

**핵심 기능:**
1. **데일리 드롭** — 매일 12시, 6시에 한정 특가 드롭. 카운트다운 + 실시간 재고
2. **소셜 공동구매** — 친구 3명 함께 사면 추가 10% 할인
3. **럭키박스** — 원가 2~3배 가치 랜덤 상품 박스

**차별화:** "엄선된 소수 + FOMO 메카닉" 게임화 쇼핑. 스니커즈 드롭 문화 일반화.

## 3. ReWear (리웨어)

> AI가 중고 의류 상태 자동 등급 분류 + 간편 판매·구매 패션 리커머스

| 항목 | 내용 |
|------|------|
| **타겟** | 옷장 정리 2030 여성, 가성비 패션 대학생, 지속가능 패션 소비자 |
| **수익 모델** | 판매 수수료 20%, 큐레이션 구독 $4.99/월, 브랜드 리사이클 B2B |
| **기술 스택** | Flutter, Supabase, Vision AI, 자체 가격 예측 모델, CJ대한통운 API |

**핵심 기능:**
1. **AI 상태 등급** — 사진 4장 → S/A/B/C 자동 분류 + 적정가격 제안
2. **원터치 판매** — 수거 택배 → 검수 → 판매 대행 → 입금. 사진만 찍으면 끝
3. **스타일 매칭** — 구매자 체형·스타일 기반 중고 의류 AI 추천

**차별화:** "사진만 찍으면 나머지는 알아서" 풀서비스 중고 패션. ThredUp 한국판 + AI.

## 4. ShopCast (샵캐스트)

> 인플루언서 실시간 라이브로 상품 소개 + 시청자 바로 구매 소셜 라이브커머스

| 항목 | 내용 |
|------|------|
| **타겟** | 라이브 쇼핑 즐기는 30~50대, 라이브 판매 크리에이터 |
| **수익 모델** | 판매 수수료 8~12%, 셀러 구독 $19.99/월, 광고 슬롯 |
| **기술 스택** | Flutter, Supabase, Agora/Mux, Stripe/토스, GPT-4o |

**핵심 기능:**
1. **라이브 쇼핑 스트리밍** — 실시간 방송 + 원탭 구매 + 채팅
2. **쇼트폼 쇼핑** — 30초~1분 숏폼 리뷰 + 구매 링크
3. **AI 하이라이트 클립** — 1시간 라이브 → 핵심 클립 자동 생성

**차별화:** 개인 크리에이터 누구나 쉽게 라이브 판매. "크리에이터 퍼스트" 플랫폼.

## 5. GiftGenie (기프트지니)

> AI가 상대 취향 분석 → 완벽한 선물 추천 + 배송까지 원스톱

| 항목 | 내용 |
|------|------|
| **타겟** | 선물 고르기 어려운 모든 사람, 기념일 많은 20~40대 직장인 |
| **수익 모델** | 선물 판매 수수료 15%, 프리미엄 $3.99/월, 브랜드 노출비 |
| **기술 스택** | Flutter, Supabase, GPT-4o, 배송 API, Stripe/토스 |

**핵심 기능:**
1. **AI 선물 추천** — 나이, 성별, 관계, 취미, 예산 → 3~5개 추천
2. **기념일 리마인더** — 생일/기념일 D-7, D-3 선물 추천 푸시
3. **원클릭 선물 + 메시지카드** — AI 감성 메시지 자동 생성 → 포장 배송

**차별화:** "AI 큐레이션 + 감성 메시지"로 "진짜 좋아할 선물" 경험.

## 6. FreshBox (프레시박스)

> 동네 농산물 직거래 구독 — 매주 제철 농산물 박스 농장 직배송

| 항목 | 내용 |
|------|------|
| **타겟** | 신선·안전한 먹거리 관심 3040 가정, 로컬푸드 건강 소비자 |
| **수익 모델** | 구독 박스 마진 20~30%, 단품 판매, 프리미엄 유기농 박스 |
| **기술 스택** | Next.js, Flutter, Supabase, Stripe/토스 |

**핵심 기능:**
1. **주간 프레시박스** — 매주 제철 채소/과일 농장 직배송, 무농약/유기농 엄선
2. **농부 스토리** — 재배 과정, 농부 소개, 산지 라이브 투어
3. **레시피 연동** — 이번 주 박스 재료 → AI 레시피 자동 추천

**차별화:** "소규모 농가 직거래 + 스토리텔링 + 구독". CSA 디지털 버전.

## 7. PriceRadar (프라이스레이더)

> 원하는 상품 최저가 실시간 추적 + 목표가 알림 + AI 할인 예측

| 항목 | 내용 |
|------|------|
| **타겟** | 알뜰 쇼핑 소비자, 고가 전자제품 구매 계획 중인 사람 |
| **수익 모델** | 쇼핑몰 리퍼럴 CPA, 프리미엄 $2.99/월, 쇼핑몰 광고 |
| **기술 스택** | Next.js + Vercel, Supabase, Puppeteer/Playwright, GPT-4o-mini |

**핵심 기능:**
1. **멀티몰 가격 비교** — 쿠팡, 11번가, G마켓, 네이버 등 10+ 쇼핑몰
2. **가격 추적 & 알림** — 목표가 이하 시 알림, 역대 최저가 그래프
3. **할인 예측** — AI 과거 패턴 분석 → 구매 타이밍 추천

**차별화:** "가격 추적 + AI 할인 예측" → "언제 사야 가장 싼지" 알려주는 어드바이저.

## 8. ReviewTrust (리뷰트러스트)

> AI가 가짜 리뷰 필터링 + 진짜 구매 경험만 모아주는 신뢰 리뷰 플랫폼

| 항목 | 내용 |
|------|------|
| **타겟** | 리뷰 조작에 지친 온라인 쇼핑 소비자 |
| **수익 모델** | 브랜드 리뷰 분석 B2B, 구매 링크 수수료, 프리미엄 $2.99/월 |
| **기술 스택** | Next.js, Supabase, GPT-4o, 크롤링 파이프라인 |

**핵심 기능:**
1. **AI 리뷰 진위 검증** — 텍스트 패턴, 작성자 이력, 사진 메타데이터 분석
2. **크로스 플랫폼 통합** — 쿠팡, 네이버, 구글, 인스타 리뷰 한곳에
3. **리뷰 요약** — 수백 개 리뷰 → "장점 TOP3, 단점 TOP3" 한눈에

**차별화:** "가짜 리뷰 필터링" 핵심. 한국 온라인 쇼핑 고질적 문제 해결.

## 9. BarterHub (바터허브)

> 돈 대신 물물교환 — 물건이나 서비스를 교환하는 교환 경제 플랫폼

| 항목 | 내용 |
|------|------|
| **타겟** | 미니멀리스트, 지속가능 소비 관심 2030, 대학생/사회초년생 |
| **수익 모델** | 프리미엄 매칭 $3.99/월, 교환 보험, 프로모션 광고 |
| **기술 스택** | Flutter, Supabase, Vision AI, 블록체인(바터코인) |

**핵심 기능:**
1. **교환 매칭** — "내 카메라 ↔ 너의 기타" AI 가치 추정 공정 교환
2. **서비스 교환** — "영어 과외 2시간 ↔ 요리 레슨 2시간" 스킬 교환
3. **교환 크레딧** — 직접 매칭 안 될 때 "바터코인" 간접 교환

**차별화:** "교환" 중심 순수 물물교환 경제. Z세대 대안 경제 관심.

## 10. CraftLocal (크래프트로컬)

> 동네 수제 작가/공방의 핸드메이드 상품 로컬 크래프트 마켓플레이스

| 항목 | 내용 |
|------|------|
| **타겟** | 수제 제품 좋아하는 2030 여성, 동네 공방 작가 |
| **수익 모델** | 판매 수수료 12%, 작가 구독 $9.99/월, 원데이 클래스 수수료 |
| **기술 스택** | Next.js, Flutter, Supabase, Mapbox |

**핵심 기능:**
1. **작가 스토리 프로필** — 작업 과정, 철학, 영상 스토리텔링
2. **동네 공방 지도** — 주변 공방/작가 탐색, 원데이 클래스 예약
3. **커스텀 주문** — 작가와 1:1 채팅으로 맞춤 주문

**차별화:** "온라인 구매 + 오프라인 체험(공방 방문, 원데이 클래스)" O2O 결합.
