## 1. StudyBuddy AI (스터디버디 AI)

> 교재/노트 업로드 → AI가 맞춤 퀴즈·요약·플래시카드 자동 생성

| 항목 | 내용 |
|------|------|
| **타겟** | 시험 준비 대학생, 자격증 수험생, 공무원 시험 준비생 |
| **수익 모델** | 프리미엄 $6.99/월, 학원/교육기관 B2B, 교재 출판사 제휴 |
| **기술 스택** | Next.js, Flutter, Supabase, GPT-4o, OCR |

**핵심 기능:**
1. **AI 콘텐츠 변환** — PDF/사진/노트 → 요약 + 퀴즈 + 플래시카드 자동 생성
2. **간격 반복 학습(SRS)** — 에빙하우스 망각곡선 기반 복습 스케줄 자동 설계
3. **약점 분석 대시보드** — 틀린 문제 분석 → 취약 주제 집중 문제 자동 출제

**차별화:** "업로드만 하면 AI가 알아서" + 한국 시험 형식(수능, 공무원, 자격증) 최적화.

## 2. LangPlay (랭플레이)

> AI 캐릭터와 상황극(롤플레이)으로 외국어 연습하는 게이미피케이션 어학 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 회화 연습 기회 없는 학습자, 재미있게 공부 원하는 20~30대 |
| **수익 모델** | 프리미엄 $8.99/월, 연간 $69.99, 기업 어학연수 B2B |
| **기술 스택** | Flutter, Supabase, GPT-4o, Whisper, TTS API |

**핵심 기능:**
1. **AI 롤플레이 시나리오** — "카페 주문", "면접 연습", "공항 길찾기" 실생활 대화
2. **실시간 발음 피드백** — 음성 인식 + 원어민 발음 비교, 문장별 점수
3. **레벨업 & 스토리 모드** — 미션 완료 → 경험치, RPG 스토리 + 매일 미션

**차별화:** "진짜 상황에서 진짜 대화" AI 구현. 한국인 취약 발음(r/l, th) 특화 교정.

## 3. SkillBadge (스킬뱃지)

> 1~4주 단기 과정 + 블록체인 인증 마이크로크레덴셜 실무 스킬 학습 플랫폼

| 항목 | 내용 |
|------|------|
| **타겟** | 이직/전직 준비 직장인, 포트폴리오 필요 주니어 |
| **수익 모델** | 과정당 $19~49, 월정액 $14.99/월, 기업 교육 B2B, 채용 연계 |
| **기술 스택** | Next.js, Supabase, Polygon, Monaco Editor, GPT-4o |

**핵심 기능:**
1. **바이트 사이즈 코스** — 1~4주 완성, 하루 30분 설계
2. **프로젝트 기반 평가** — 실제 프로젝트 제출 + AI 리뷰 → 합격 시 뱃지
3. **블록체인 크레덴셜** — 검증 가능 디지털 배지, 링크드인/이력서 추가

**차별화:** "짧고 실무적 + 블록체인 인증". 기업 HR이 검증 가능한 크레덴셜.

## 4. MathVerse (매쓰버스)

> 수학 문제 사진 → AI 단계별 풀이 + 개념 시각화 수학 튜터

| 항목 | 내용 |
|------|------|
| **타겟** | 수학 어려운 중·고등학생, 수능/내신 준비생, 학부모 |
| **수익 모델** | 프리미엄 $7.99/월, 학원 단체 B2B, 교육청/학교 납품 |
| **기술 스택** | Flutter, Supabase, GPT-4o, MathJax/Desmos, Vision AI |

**핵심 기능:**
1. **카메라 풀이** — 문제 사진 → AI 인식 → 칠판 애니메이션으로 단계별 풀이
2. **개념 시각화** — 함수 그래프, 도형, 벡터 인터랙티브 시각화
3. **취약 단원 집중 학습** — 틀린 유형 분석 → 관련 개념 영상 + 유사 문제 추천

**차별화:** "왜 이렇게 푸는지" 개념 시각화 + 취약점 기반 맞춤 학습. 한국 교육과정 반영.

## 5. ReadTogether (리드투게더)

> 같은 책 읽는 사람들 모여 토론 + 독서 습관 만드는 소셜 독서 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 독서 습관 만들고 싶은 20~30대, 북클럽 관심 직장인 |
| **수익 모델** | 프리미엄 $4.99/월, 도서 구매 제휴, 작가 라이브 세션 티켓 |
| **기술 스택** | Flutter, Supabase, GPT-4o-mini, 알라딘/교보 API |

**핵심 기능:**
1. **AI 북클럽 매칭** — 책 + 독서 속도 기반 5~10명 온라인 북클럽 자동 구성
2. **챕터별 토론 보드** — 완독 시 토론 질문 해금, AI 주제 제안
3. **독서 트래커 & 리포트** — 연간 독서량, 장르 분포, 완독률 공유 카드

**차별화:** "함께 읽고 토론하는" 소셜 독서. 한국 독서모임 문화 디지털화.

## 6. ParentCoach (페런트코치)

> 자녀 연령별 발달 가이드 + AI 양육 상담 + 가정학습 활동

| 항목 | 내용 |
|------|------|
| **타겟** | 0~12세 자녀 둔 부모, 첫 아이 육아 30~40대 |
| **수익 모델** | 프리미엄 $5.99/월, 육아용품 제휴, 소아과/발달센터 연계 |
| **기술 스택** | Flutter, Supabase, GPT-4o, 자체 아동발달 DB |

**핵심 기능:**
1. **발달 단계별 가이드** — 월령/나이별 이정표, 추천 활동, 주의사항
2. **AI 양육 상담** — "밥을 안 먹어요" → 아동발달 전문가 수준 AI 조언
3. **가정학습 액티비티** — "종이접기로 공간 지각력 키우기" 연령별 놀이 추천

**차별화:** "발달 가이드 + AI 상담 + 활동 추천" 통합. 한국 부모 교육열 + 불안감 해소.

## 7. ClassMate AI (클래스메이트 AI)

> 교사 위한 AI 수업 보조 — 수업자료 생성, 채점 자동화, 학생 이해도 분석

| 항목 | 내용 |
|------|------|
| **타겟** | 초·중·고 교사, 학원 강사, 온라인 강의 제작자 |
| **수익 모델** | 교사 개인 $9.99/월, 학교 단체, 교육청 B2G, 학원 B2B |
| **기술 스택** | Next.js, Supabase, GPT-4o, OCR, Chart.js |

**핵심 기능:**
1. **AI 수업자료 생성** — 단원명 → 교안, 활동지, 퀴즈, PPT 자동 생성
2. **자동 채점 & 피드백** — 서술형 답안 AI 채점 + 개인별 피드백
3. **학생 이해도 히트맵** — 반 전체 단원별 이해도 시각화

**차별화:** 한국 2022 개정 교육과정 맞춤 설계. 교사 업무 부담 실질적 감소.

## 8. DebateArena (디베이트아레나)

> AI 상대 또는 실제 유저와 실시간 토론 대결 — 논리력·표현력 학습

| 항목 | 내용 |
|------|------|
| **타겟** | 논술/면접 준비생, 토론 동아리, 비판적 사고력 키우려는 20~30대 |
| **수익 모델** | 프리미엄 $6.99/월, 교육기관 B2B, 대회 참가비 |
| **기술 스택** | Flutter, Supabase, GPT-4o, WebRTC |

**핵심 기능:**
1. **AI 디베이터** — AI가 상대편 되어 논리적 반박, 난이도 조절
2. **실시간 매칭 토론** — 같은 주제 유저와 1:1 텍스트/음성 토론
3. **AI 토론 코칭** — 논리 구조, 근거 활용, 반박 효과성 분석 리포트

**차별화:** "격투게임 식 토론" 게이미피케이션. 한국 논술/면접 중시 교육 문화.

## 9. CodeKids (코드키즈)

> 5~12세 어린이 블록 코딩 + 게임 만들기로 컴퓨팅 사고력 교육

| 항목 | 내용 |
|------|------|
| **타겟** | 코딩 교육 관심 학부모의 5~12세 자녀, 초등 방과후 코딩 교실 |
| **수익 모델** | 프리미엄 $5.99/월, 가족 $9.99/월, 학교/학원 B2B |
| **기술 스택** | Flutter, Supabase, Blockly, GPT-4o-mini |

**핵심 기능:**
1. **스토리 기반 블록 코딩** — 캐릭터 모험 = 코딩 학습, 드래그&드롭
2. **나만의 게임 만들기** — 배운 코딩으로 게임/애니메이션 제작 + 공유
3. **AI 힌트 시스템** — 답 안 주고 단계별 힌트, 소크라테스식

**차별화:** "스토리 + 게이미피케이션 + AI 힌트" 자기주도형 코딩 교육.

## 10. CareerPath (커리어패스)

> AI가 경력·스킬·적성 분석 → 최적 커리어 로드맵 + 교육 과정 추천

| 항목 | 내용 |
|------|------|
| **타겟** | 진로 고민 대학생, 이직/전직 3~7년차, 경력 단절 복귀자 |
| **수익 모델** | 프리미엄 $9.99/월, 교육 플랫폼 리퍼럴, 기업 인재개발 B2B |
| **기술 스택** | Next.js, Flutter, Supabase, GPT-4o, 채용 데이터 크롤링 |

**핵심 기능:**
1. **AI 커리어 진단** — 이력서 + 적성 테스트 → 적합 직무 TOP5 + 스킬 갭
2. **맞춤 로드맵** — "프론트엔드→PM 전환" 필요 스킬·자격증·경험 타임라인
3. **교육 큐레이션** — 로드맵에 맞는 강의(유데미/인프런/코세라) 추천 + 진도 추적

**차별화:** "진단→설계→교육→취업" 커리어 여정 전체 관리. 한국 높은 이직률+리스킬링 수요.
