## 1. FlowState (플로우스테이트)

> 딥워크 세션 설계 + 방해 요소 차단 + 집중 데이터 분석 초집중 생산성 앱

| 항목 | 내용 |
|------|------|
| **타겟** | 집중력 저하 개발자, 디자이너, 작가, 재택근무자 |
| **수익 모델** | 프리미엄 $5.99/월, 기업 웰빙 B2B, 생산성 코칭 연동 |
| **기술 스택** | Flutter, Supabase, 스크린타임 API, HealthKit |

**핵심 기능:**
1. **딥워크 세션 디자이너** — 업무별 최적 집중 시간, 포모도로 커스텀 + 휴식 가이드
2. **스마트 방해 차단** — SNS/뉴스 차단, 알림 무음화, "긴급 연락만" 화이트리스트
3. **집중 분석 대시보드** — 일/주/월 추이, 방해 요인 TOP3, 최적 시간대

**차별화:** "차단+분석+최적화" 데이터 기반 딥워크. 개인 집중 패턴 학습.

## 2. TaskFlow AI (태스크플로우 AI)

> 자연어로 업무 말하면 AI가 할 일 분해·우선순위·일정 배치까지

| 항목 | 내용 |
|------|------|
| **타겟** | 할 일 너무 많은 직장인, PM, 프리랜서 |
| **수익 모델** | 프리미엄 $7.99/월, 팀 $14.99/인/월, 기업 B2B |
| **기술 스택** | Flutter, Next.js, Supabase, GPT-4o, Google Calendar API |

**핵심 기능:**
1. **자연어 태스크 입력** — "금요일까지 기획서" → 하위 태스크 분해
2. **AI 우선순위 엔진** — 긴급도, 중요도, 소요시간, 에너지 고려 자동 순위
3. **캘린더 자동 블로킹** — 빈 시간에 태스크 자동 배치, 에너지 패턴 반영

**차별화:** "말만 하면 AI가 다 정리" 제로 프릭션. 한국어 자연어 처리 특화.

## 3. MeetingZero (미팅제로)

> 불필요한 회의 줄이고 비동기 커뮤니케이션으로 대체하는 팀 생산성 도구

| 항목 | 내용 |
|------|------|
| **타겟** | 회의 과다 팀 리더, 매니저, 하이브리드/리모트 팀 |
| **수익 모델** | 팀 $8/인/월, 기업 라이선스, Slack/Teams 연동 프리미엄 |
| **기술 스택** | Next.js, Supabase, Slack/Teams API, GPT-4o |

**핵심 기능:**
1. **회의 필요성 검증** — "Loom 영상 + 비동기 코멘트로 대체 가능" AI 판단
2. **비동기 스탠드업** — 매일 3줄 비동기 공유, 리더 대시보드
3. **회의 ROI 대시보드** — 주간 회의 시간, 참석 인원, 액션 아이템 추적

**차별화:** "회의를 줄이는 것" 자체가 목적. 한국 과도한 회의 문화 개선.

## 4. DocuMind (도큐마인드)

> 사내 문서/위키를 AI 학습 → 자연어 질문 → 즉시 답변 사내 지식 검색 엔진

| 항목 | 내용 |
|------|------|
| **타겟** | 사내 문서 흩어진 중소기업, 온보딩 신입사원, IT/CS 팀 |
| **수익 모델** | 팀 $12/인/월, 온프레미스 라이선스, 문서 수 종량제 |
| **기술 스택** | Next.js, Supabase, RAG + pgvector, Notion/Google Drive API, GPT-4o |

**핵심 기능:**
1. **원클릭 지식 베이스** — Notion/Google Drive/Confluence 자동 인덱싱
2. **AI 사내 검색** — "휴가 신청 어떻게?" → 정확한 답변 + 원문 링크
3. **자동 FAQ 생성** — 자주 묻는 질문 AI 분석 → FAQ 자동 생성·업데이트

**차별화:** "물어보면 바로 답" ChatGPT급 사내 지식 비서. 온보딩 50% 단축.

## 5. HabitOS (해빗OS)

> 행동 과학 기반 습관 설계 + AI 넛지 + 커뮤니티 책임 파트너

| 항목 | 내용 |
|------|------|
| **타겟** | 매번 작심삼일인 20~40대, 운동/독서/명상 습관 |
| **수익 모델** | 프리미엄 $4.99/월, 기업 웰빙 B2B, 습관 콘텐츠 구독 |
| **기술 스택** | Flutter, Supabase, GPT-4o-mini, HealthKit |

**핵심 기능:**
1. **습관 스태킹 디자이너** — 제임스 클리어 원칙 기반 AI 설계
2. **AI 넛지 & 동기부여** — 포기 타이밍에 맥락 알림, 행동 경제학 기법
3. **어카운터빌리티 파트너** — 2인 1조 매칭, 서로 체크인 + 사회적 압력

**차별화:** 행동 과학 + AI + 소셜 압력 3중 구조로 습관 형성률 과학적 향상.

## 6. WriterFlow (라이터플로우)

> 작가·블로거 위한 집중 글쓰기 환경 + AI 편집 어시스턴트 + 퍼블리싱 연동

| 항목 | 내용 |
|------|------|
| **타겟** | 블로거, 뉴스레터 작가, 웹소설 작가, 콘텐츠 마케터 |
| **수익 모델** | 프리미엄 $6.99/월, 작가 커뮤니티 프로 $12.99/월 |
| **기술 스택** | Next.js, Supabase, Tiptap/ProseMirror, GPT-4o |

**핵심 기능:**
1. **미니멀 에디터** — 풀스크린 글쓰기, 마크다운 + 글자수/읽기 시간
2. **AI 편집 어시스턴트** — 맞춤법, 문체 일관성, 수식어, 문장 길이 수정 제안
3. **원클릭 퍼블리싱** — 네이버 블로그, 브런치, 미디엄, 스티비에 한번에 발행

**차별화:** "글 쓰는 사람"만 위한 최적 환경 + 한국 퍼블리싱 플랫폼 연동.

## 7. InboxZero (인박스제로)

> 이메일·슬랙·카톡·문자 모든 메시지를 AI가 정리하는 통합 커뮤니케이션 허브

| 항목 | 내용 |
|------|------|
| **타겟** | 10개+ 채널 메시지 쏟아지는 팀 리더, 매니저, 영업직 |
| **수익 모델** | 프리미엄 $9.99/월, 팀 $19.99/인/월, 기업 B2B |
| **기술 스택** | Next.js, Flutter, Supabase, Gmail/Slack/Kakao API, GPT-4o |

**핵심 기능:**
1. **유니파이드 인박스** — 이메일/슬랙/카톡/문자 하나의 뷰
2. **AI 트리아지** — 긴급/중요/참고/스팸 자동 분류 + 답장 필요 태그
3. **스마트 답장 제안** — 맥락 파악 → 3가지 답장 초안 → 원클릭 발송

**차별화:** "모든 메시지를 하나로" + 한국 카카오톡 업무 메시지 문화 지원.

## 8. TemplateHQ (템플릿HQ)

> 기획서, 제안서, 보고서, 이력서 등 비즈니스 문서 AI 맞춤 생성

| 항목 | 내용 |
|------|------|
| **타겟** | 문서 작성 잦은 직장인, 스타트업 대표, 프리랜서, 취준생 |
| **수익 모델** | 프리미엄 $7.99/월, 기업 브랜드팩 $29/월, 템플릿 마켓 수수료 |
| **기술 스택** | Next.js, Supabase, GPT-4o, Tiptap, Puppeteer |

**핵심 기능:**
1. **AI 문서 생성** — "시리즈A 투자 제안서" → 구조화된 초안 자동 생성
2. **500+ 템플릿 라이브러리** — 한국 비즈니스 관행 맞춤 형식
3. **공동 편집 & 브랜딩** — 실시간 협업, 로고·색상 적용, PDF/PPTX 내보내기

**차별화:** "모든 비즈니스 문서를 AI가 맞춤 생성" 원스톱. 한국식 문서 양식 지원.

## 9. RetroBoard (레트로보드)

> 팀 회고(Retrospective) AI 퍼실리테이팅 + 액션 아이템 추적 애자일 도구

| 항목 | 내용 |
|------|------|
| **타겟** | 스크럼/애자일 팀, 스타트업 팀 리더, PM |
| **수익 모델** | 팀 $6/인/월, 기업 라이선스, Jira/Linear 연동 프리미엄 |
| **기술 스택** | Next.js, Supabase, GPT-4o-mini, Jira/Linear API |

**핵심 기능:**
1. **회고 템플릿** — Start/Stop/Continue, 4Ls 등 다양한 프레임워크
2. **AI 퍼실리테이터** — 비슷한 의견 그룹핑, 핵심 주제 도출, 토론 질문
3. **액션 아이템 추적** — 개선 사항 → Jira/Linear 자동 티켓 + 다음 회고 리뷰

**차별화:** "AI 퍼실리테이션 + 액션 추적"으로 회고 실질적 개선. 형식적 회고 방지.

## 10. AutoReport (오토리포트)

> 주간/월간 업무 보고서 AI 자동 작성 — 캘린더·메일·업무도구 데이터 수집

| 항목 | 내용 |
|------|------|
| **타겟** | 매주 보고서 1~2시간 쓰는 직장인, 팀 리더, 중간 관리자 |
| **수익 모델** | 프리미엄 $8.99/월, 팀 $14.99/인/월, 기업 B2B |
| **기술 스택** | Next.js, Supabase, Google/Jira/Slack API, GPT-4o |

**핵심 기능:**
1. **데이터 자동 수집** — Calendar/Jira/Slack/Gmail 활동 데이터 수집
2. **AI 보고서 생성** — "이번 주 한 일, 성과, 다음 주 계획" 초안 자동
3. **커스텀 템플릿** — 회사/팀별 형식 설정, 매주 같은 형식 자동 생성

**차별화:** "데이터 수집부터 초안까지 자동화" 주당 1~2시간 절약. 한국식 주간/월간보고 맞춤.
