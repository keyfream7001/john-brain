# 🚀 "Hey Siri, AI한테 물어봐" - 나만의 AI 비서를 Siri와 연결한 이야기

> 아이폰에서 Siri로 내 AI 비서를 호출할 수 있다면? 오늘 그걸 실제로 만들었습니다.

---

## 🎬 시작: 왜 Siri와 AI를 연결하려 했나?

ChatGPT 앱도 있고, Claude 앱도 있는데 왜 굳이?

제가 원한 건 **진짜 개인 비서**였어요.

- 내 파일에 접근하고
- 내 일정을 알고
- 내 프로젝트를 이해하는

그런 AI를 만들었는데, 매번 컴퓨터 앞에서만 쓸 수 있다니 아쉽더라고요.

그래서 생각했죠. **"Hey Siri"로 부르면 어떨까?**

---

## 🔧 과정: 생각보다 간단했다

### 1단계: VPN 연결 (Tailscale)

먼저 어디서든 내 컴퓨터에 접근할 수 있어야 해요.

```
아이폰 → 인터넷 → Tailscale VPN → 내 PC
```

Tailscale은 무료고, 설정도 앱 설치만 하면 끝!

### 2단계: API 엔드포인트 확인

제 AI(웬디)는 Clawdbot이라는 프레임워크 위에서 돌아가는데, HTTP API를 제공해요.

```
https://wendy.tail9938b1.ts.net/v1/chat/completions
```

### 3단계: iOS 단축어 설정

단축어 앱에서:
1. **텍스트** → 질문 받기
2. **URL 가져오기** → POST 요청
3. **텍스트** → 응답 추출
4. **말하기** → TTS로 읽어주기

**끝!**

---

## ✨ 결과: 이렇게 됩니다

> **나**: "Hey Siri, 웬디한테 물어봐"
> 
> **Siri**: "뭘 물어볼까요?"
> 
> **나**: "오늘 할 일 뭐야?"
> 
> **웬디**: "오늘은 LottoMate 앱 버그 수정이랑, 저녁에 친구 약속 있어요!"

내 할 일 목록, 일정, 메모에 모두 접근 가능한 진짜 비서!

---

## 💡 배운 점들

### 1. 속도가 중요하면 경량 모델

Siri 응답은 빨라야 자연스럽잖아요.

그래서 Claude Opus 대신 **Haiku 모델**을 썼어요. 응답이 1-2초 안에 와서 대화처럼 느껴집니다.

### 2. VPN은 생각보다 쉽다

Tailscale 덕분에 복잡한 포트 포워딩이나 공유기 설정 없이 바로 연결!

집 밖에서도 내 PC의 AI에 접속할 수 있어요.

### 3. 음성 UI는 다르다

텍스트로 길게 쓰는 것과 말로 듣는 건 달라요.

AI 응답을 짧고 핵심만 담도록 프롬프트를 조정했습니다.
