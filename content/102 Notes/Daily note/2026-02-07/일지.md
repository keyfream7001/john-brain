## 💻 Flutter 앱 개발

### LottoMate Overflow 에러 수정

**요청**: 홈 화면과 번호 생성 화면의 UI 깨짐 수정

**문제점**:
1. 홈 화면 카드 - "BOTTOM OVERFLOWED BY 5.3 PIXELS"
2. 번호 생성 화면 로또공 - "RIGHT OVERFLOWED BY 26 PIXELS"
3. 최신 당첨번호 카드 - "OVERFLOWED BY 53 PIXELS"

**해결 방법**:
```dart
// Before: Row가 화면 밖으로 넘침
Row(
  children: [...lottoBalls],
)

// After: FittedBox로 자동 축소
FittedBox(
  fit: BoxFit.scaleDown,
  child: Row(
    mainAxisSize: MainAxisSize.min,
    children: [...lottoBalls],
  ),
)
```

**결과**: 모든 화면 크기에서 UI 정상 표시

**배운점**:
- `FittedBox`는 반응형 UI의 핵심 위젯
- `mainAxisSize: MainAxisSize.min`으로 불필요한 공간 제거
- 고정 크기보다 비율 기반 레이아웃이 유연함

**강의 포인트**:
- 🎯 Flutter Overflow 에러 디버깅 방법
- 🎯 FittedBox, Flexible, Expanded 활용법
- 🎯 반응형 UI 설계 원칙
