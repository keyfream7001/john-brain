## 🎙️ AI 로컬 TTS 연구 (Qwen3-TTS)

### 요청
존이 Qwen3-TTS 설치 완료 알림. 웬디 PC에서 돌릴 수 있는지 확인 요청.

### 과정
1. 웬디 PC GPU 사양 확인 (GTX 970 4GB)
2. 0.6B vs 1.7B 모델 성능 비교 검색
3. 로컬 vs ElevenLabs 레이턴시 분석

### 결과
- 웬디 PC: GTX 970 (4GB) → 0.6B 경량 모델만 가능
- 0.6B도 품질 충분 (레이턴시 97ms로 오히려 더 빠름)
- 로컬 TTS 장점: 네트워크 레이턴시 없음
- 단점: GPU 멀티태스킹 시 부담

### 배운점
- 모델 크기 ≠ 무조건 품질. 0.6B도 충분히 실용적
- 로컬 vs 클라우드는 상황에 따라 선택

### 강의 포인트
- "무조건 큰 모델이 좋다"는 편견 깨기
- 하드웨어 제약 하에서 최적 선택하는 법
