# AI 비서 웬디와 함께한 하루: 무료 API 찾아 삼만리

안녕하세요! 오늘은 제 AI 비서 '웬디'와 함께 꽤 알찬 하루를 보냈어요. AI를 일상에서 활용하면서 느낀 점들을 공유해볼게요.

---

## 🎯 오늘의 미션: API 비용 줄이기

솔직히 말하면, AI API 비용이 부담되기 시작했어요. Claude를 메인으로 쓰다 보니 토큰비가 꽤 나오더라고요. 그래서 오늘의 목표는 **무료 또는 저렴한 AI API 찾기**였습니다.

웬디에게 물었죠.

> "무료 AI API 있어?"

그랬더니 Z.AI라는 서비스를 알려주더라고요. 중국 회사인데, 무료 모델을 제공한다고!

---

## 🆓 Z.AI 발견! 진짜 무료야?

반신반의하며 가입해봤는데... 진짜였어요!

**완전 무료 모델들:**
- GLM-4.7-Flash - 일반 대화용
- GLM-4.5-Flash - 가벼운 작업용  
- GLM-4.6V-Flash - 이미지 분석까지!

가입도 간단했어요. 구글 계정으로 로그인하면 끝. API 키 발급받고 테스트해보니 잘 작동하더라고요.

**"야, 이거 대박인데?"**

...라고 좋아했는데, 문제가 있었어요.

---

## 😅 연동의 벽

제가 쓰는 OpenClaw(AI 에이전트 플랫폼)에 Z.AI를 연결하려고 했는데... "Model not allowed" 에러가 뜨더라고요.

API 직접 호출은 되는데, 플랫폼 연동이 안 되는 상황.

웬디랑 이것저것 시도해봤지만 결국 **"일단 보류"**로 결론. 나중에 OpenClaw 업데이트되면 다시 시도하기로 했어요.

> 💡 교훈: 무료라고 다 좋은 건 아니다. 호환성 체크 필수!

---

## 🔄 OpenClaw 대규모 업데이트

마침 OpenClaw 새 버전이 나왔더라고요. 2026.2.6 버전!

주요 업데이트:
- Anthropic Opus 4.6 지원
- xAI (Grok) 프로바이더 추가
- 토큰 사용량 대시보드

업데이트 자체는 쉬웠어요. `npm install -g openclaw` 한 줄이면 끝.

**그런데...**

설정 파일이 다 날아갔어요 ㅋㅋㅋ

폴더 이름이 `.clawdbot`에서 `.openclaw`로 바뀌면서, 기존 설정을 수동으로 옮겨야 했거든요. 웬디랑 열심히 파일 복사하고... 맥북 연결도 다시 하고...

> 💡 교훈: 대규모 업데이트 전엔 백업하자!
