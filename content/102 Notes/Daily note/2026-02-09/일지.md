## 1. 크롬 브라우저 릴레이 익스텐션 설치

### 목적
- AI 에이전트(웬디)가 사용자의 크롬 브라우저를 직접 제어할 수 있게 연결

### 설치 과정
1. `clawdbot browser extension install` 명령 실행
2. Chrome 설정 → 확장 프로그램 → 개발자 모드 활성화
3. "압축 해제된 확장 프로그램 로드" 클릭
4. 경로 선택: `C:\Users\phase\.clawdbot\browser\chrome-extension`

### 사용 방법
- AI에게 `profile="chrome"` 옵션으로 브라우저 제어 요청
- 실제 크롬 탭을 열고, 클릭하고, 타이핑 가능

### 활용 사례
- 웹사이트 자동 로그인
- 정보 수집 및 스크래핑
- 반복 작업 자동화

## 2. Z.AI (智谱AI) 무료 API 설정

### Z.AI 소개
- 중국의 AI 회사 (Zhipu AI)
- 무료 API 제공으로 비용 절감 가능

### 계정 생성 및 API 발급
1. https://z.ai 접속
2. Google 계정으로 로그인
3. 대시보드에서 API Key 발급
4. `.env` 파일에 `ZAI_API_KEY` 저장

### 무료 모델 정보
| 모델명 | 용도 | 가격 | 제한 |
|--------|------|------|------|
| GLM-4.7-Flash | 텍스트 생성 | 무료 | 동시 1개 |
| GLM-4.5-Flash | 텍스트 생성 | 무료 | - |
| GLM-4.6V-Flash | 이미지 분석 | 무료 | - |
| GLM-4.7 | 고급 텍스트 | $0.6/1M tokens | - |

## 4. 설정 마이그레이션

### 복사한 폴더/파일 목록
```
.clawdbot/
├── agents/          → .openclaw/agents/
├── browser/         → .openclaw/browser/
├── credentials/     → .openclaw/credentials/
├── telegram/        → .openclaw/telegram/
├── media/           → .openclaw/media/
├── memory/          → .openclaw/memory/
├── subagents/       → .openclaw/subagents/
├── .env             → .openclaw/.env
├── exec-approvals.json
└── devices/paired.json
```

### 주의사항
- 수동 복사 필요 (자동 마이그레이션 없음)
- paired.json은 노드 재연결 시 필요
