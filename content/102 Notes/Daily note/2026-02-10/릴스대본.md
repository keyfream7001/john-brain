# [릴스/숏츠 대본] AI 3명한테 동시에 코딩 시켜봄

## 버전 1: 충격 결과형 (30초)

**[0-3초]** 
*(화면: 터미널 3개 동시 실행)*
"AI 3명한테 동시에 코딩 시켜봤습니다"

**[3-8초]**
*(화면: 할 일 목록)*
"반응형 UI, 모바일 최적화, 이벤트 페이지, API까지"
"혼자 하면 일주일 걸릴 작업"

**[8-15초]**
*(화면: 각 에이전트 역할)*
"1번은 UI, 2번은 백엔드, 3번은 신기능"
"역할 나눠서 동시에 돌렸습니다"

**[15-25초]**
*(화면: 커밋 로그 스크롤)*
"결과요? 하루 만에 커밋 7개"
"심지어 로딩 UX는 알아서 이쁘게 만들어줌"

**[25-30초]**
*(화면: 완성된 웹사이트)*
"AI 에이전트 팀, 진짜 됩니다"

## 버전 2: 비포애프터형 (30초)

**[0-5초]**
*(화면: 빈 코드 에디터)*
"오늘 아침의 내 프로젝트"

**[5-10초]**
*(화면: Claude Code Agent Teams)*
"AI 에이전트 3명 소환"

**[10-20초]**
*(화면: 작업 타임랩스)*
"1번: 디자인 담당"
"2번: API 담당"
"3번: 기능 담당"

**[20-30초]**
*(화면: 완성된 화면들 빠르게 전환)*
"하루 뒤 결과"
"커밋 7개, 기능 6개 완성"
"이게 AI 멀티에이전트입니다"

## 버전 3: TMI형 (45초)

**[0-5초]**
*(셀카캠)*
"AI한테 코딩 시키는 거 다들 해보셨죠?"

**[5-10초]**
"근데 저는 한 명이 아니라 세 명한테 시켜봤어요"

**[10-20초]**
*(화면: 에이전트 구성)*
"Claude Code에 Agent Teams라는 기능이 있거든요"
"역할별로 나눠서 동시에 작업시키는 거예요"

**[20-35초]**
*(화면: 결과물들)*
"UI 담당이 반응형 만드는 동안"
"백엔드 담당은 API 짜고"
"기능 담당은 이벤트 페이지 만들고"

**[35-45초]**
*(셀카캠)*
"결과? 하루 만에 웹사이트 리뉴얼 끝"
"다음엔 5명 시켜볼까 고민 중"
