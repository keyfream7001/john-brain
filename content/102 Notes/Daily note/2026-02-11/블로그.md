# AI와 함께한 하루: 프로젝트 배포부터 마케팅까지

오늘 하루가 정말 알찼다. AI 어시스턴트 웬디와 함께 여러 프로젝트를 진행했는데, 혼자 했으면 3일은 걸렸을 일들을 하루 만에 끝냈다.

## 🎉 드디어! 1man_coding 배포 완료

몇 주간 준비해온 '만원코딩' 서비스 웹사이트를 드디어 라이브로 올렸다.

결제 시스템을 붙이는 게 가장 고민이었는데, 키움페이를 선택했다. 카카오페이, 네이버페이 다 지원하면서 수수료도 괜찮고, 무엇보다 연동이 생각보다 간단했다.

배포 과정에서 작은 문제가 있었다. GitHub에서 Vercel로 자동 배포가 안 되는 거다. 이럴 때 당황하지 않고 Deploy Hook을 사용했다. curl 명령어 한 줄이면 수동으로 배포할 수 있다는 걸 오늘 처음 알았다.

**결과물**: https://1man-coding.vercel.app

아직 다듬어야 할 부분이 있지만, 일단 라이브로 올리니까 뿌듯하다.

## 📚 시스템을 만들어야 확장할 수 있다

만원코딩 서비스를 시작하면서 깨달은 게 있다. 매번 고객마다 다르게 일하면 절대 확장이 안 된다.

그래서 오늘 시스템 문서를 만들었다:
- **고객 프로젝트 작업 방식**: 3가지 방식 중 상황에 맞게 선택
- **3시간 완성 시스템**: 타임라인, 체크리스트, 템플릿까지

웬디가 내 생각을 구조화해서 문서로 만들어주는 게 정말 좋았다. 내가 "이런 느낌으로 해야 할 것 같아"라고 말하면, 깔끔하게 정리해서 보여준다.

## 📝 스레드 마케팅 시작

만원코딩 홍보를 위해 스레드 첫 글을 작성했다.

> "제품촬영 스튜디오 7년, 개발자 3년차. 유명해지고 싶다."

이 한 줄이 훅이다. 진심을 담은 자기소개가 가장 강력한 마케팅이라고 생각한다.

## 🔧 다음 프로젝트: ResumAI

자소서 첨삭 서비스 ResumAI도 90%까지 왔다. Supabase 무료 한도 문제로 잠깐 막혔는데, 내일 정리하면 바로 배포할 수 있을 것 같다.

기술 스택은 간단하게:
- 결제: 키움페이 (1man_coding 설정 재활용)
- AI: Gemini API
- DB: Supabase
- 배포: Vercel

검증된 스택을 재사용하니까 개발 속도가 확 빨라진다.

## 💡 오늘의 교훈

**1. AI는 생산성 부스터다**
특히 문서화, 코드 수정, 마케팅 문구 작성에서 AI 어시스턴트의 도움이 컸다. 내가 방향을 잡으면 AI가 실행을 도와준다.

**2. 막히면 다른 방법을 찾아라**
GitHub 자동 배포가 안 되면 Deploy Hook, Supabase 한도가 차면 프로젝트 정리. 항상 우회로가 있다.

**3. 시스템이 곧 자산이다**
문서화해둔 워크플로우는 나중에 반복해서 사용할 수 있다. 오늘 1시간 투자해서 만든 시스템이 앞으로 몇십 시간을 절약해줄 것이다.
