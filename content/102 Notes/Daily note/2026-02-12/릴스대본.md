# 📱 [릴스/숏츠 대본] AI 에이전트 2마리 = MVP 2개

## 영상 정보
- **길이**: 30-60초
- **플랫폼**: 인스타 릴스, 유튜브 숏츠, 틱톡
- **스타일**: 빠른 컷 + 텍스트 오버레이

---

## 버전 1: 충격 & 결과 중심 (30초)

**[0-3초]**
화면: 터미널에 명령어 입력
자막: "AI한테 앱 2개 동시에 시켰더니..."

**[3-8초]**
화면: 에이전트 2개 동시에 돌아가는 모습
자막: "에이전트 2마리가 각각 다른 프로젝트 시작"

**[8-15초]**
화면: 타임랩스로 30분 지나가는 효과
자막: "30분 후..."

**[15-25초]**
화면: 완성된 두 프로젝트 빠르게 전환
자막: "MVP 2개 완성 (에러 0개)"

**[25-30초]**
화면: 나 등장 (카메라 보며)
대사: "기획서만 잘 써두면 AI가 다 만들어줌"

---

## 버전 2: How-to 중심 (45초)

**[0-5초]**
화면: 문제 상황 (앱 2개 필요)
자막: "앱 2개 급하게 필요한데 시간이 없을 때"

**[5-15초]**
화면: PLAN.md 작성 장면
자막: "1. 기획서부터 상세하게 작성"

**[15-25초]**
화면: Claude Code 명령어
자막: "2. 에이전트 팀으로 병렬 개발 지시"

**[25-35초]**
화면: 두 에이전트 동시 작업
자막: "3. AI가 동시에 2개 프로젝트 개발"

**[35-45초]**
화면: 완성된 결과물 + 빌드 성공
자막: "결과: 30분 만에 MVP 2개 완성!"

---

## 버전 3: Before/After (30초)

**[0-5초]**
화면: 지친 표정의 개발자 이미지
자막: "예전: 앱 1개 만드는 데 일주일"

**[5-10초]**
화면: 전환 효과 (스와이프)

**[10-20초]**
화면: Claude Code 에이전트 팀 실행
자막: "지금: AI 에이전트 2마리 소환"

**[20-30초]**
화면: 완성된 두 앱 나란히
자막: "30분에 앱 2개 완성 ㄷㄷ"
