# 🚀 AI 에이전트 2마리로 30분 만에 MVP 2개 만들기

안녕하세요! 오늘은 정말 신기한 경험을 공유하려고 해요.

**Claude Code의 에이전트 팀 기능**으로 30분 만에 MVP 2개를 동시에 만들어봤습니다.

---

## 💡 아이디어는 2개, 손은 2개... 아니 0개?

만원코딩 서비스를 위해 두 가지 도구가 필요했어요:

1. **EasyAPI** - 한국에서 AI API를 쉽게 쓸 수 있는 프록시 서비스
2. **10고개 앱 빌더** - 질문 10개로 앱 아이디어를 구체화하는 도구

둘 다 중요한데, 하나씩 만들자니 시간이 너무 오래 걸리고...

그래서 **둘 다 동시에** 시켜봤습니다.

---

## 🤖 에이전트 팀 소환!

Claude Code에 이렇게 명령했어요:

> "두 프로젝트 동시에 개발 시작해. 에이전트 팀으로 병렬 진행."

그랬더니...

```
├─ Agent 1: Build EasyAPI MVP Next.js app
│  ⎿ Running in background
└─ Agent 2: Build 10고개 app builder MVP
   ⎿ Running in background
```

**2마리의 AI 에이전트**가 각각 다른 프로젝트를 맡아서 동시에 코딩을 시작했습니다!

---

## ⏱️ 30분 후의 결과

### EasyAPI가 만들어낸 것들
- 로그인/회원가입 페이지
- 대시보드 전체 레이아웃
- API 키 관리 페이지
- 크레딧 충전 페이지
- Playground (API 테스트)
- 사용량 통계 페이지

### 10고개 앱 빌더가 만들어낸 것들
- Q1~Q10 질문 컴포넌트
- 4가지 템플릿 (쇼핑몰, 예약, 커뮤니티, 기본)
- 실제 아이폰처럼 생긴 폰 목업 미리보기
- 색상 선택하면 전체 UI가 바뀌는 동적 컬러 시스템
- Zustand로 상태 관리까지

**TypeScript 에러 없이 빌드 성공!**

---

## 🎯 이게 가능한 이유

1. **기획서를 먼저 만들어뒀음** - 각 프로젝트별 상세 PLAN.md 파일
2. **독립적인 작업** - 두 프로젝트가 서로 의존하지 않음
3. **dangerously-skip-permissions** - 권한 확인 없이 자동 실행

특히 **상세한 기획서**가 있으면 에이전트가 훨씬 정확하게 작업해요.

## 💬 마무리

AI 에이전트가 코드를 "대신" 짜주는 시대는 이미 왔어요.

이제 중요한 건:
- **무엇을 만들지** 명확하게 정의하기
- **어떻게 지시할지** 효과적으로 소통하기

코딩 실력보다 **기획력과 소통 능력**이 더 중요해지는 시대.

여러분도 AI 에이전트 팀을 활용해보세요!
