# 실제 포스팅 내용 (바로 복붙용) - 기존 버전

## 1) AI로 앱 만드는 시대 끝

나는 만원코딩.
요즘 "AI로 앱 만들어봤다" 글 많지?
근데 대부분 거기서 멈춰.

나는 다음 스텝이 궁금했어.
**앱을 만드는 '시스템'을 만들 수는 없을까?**

그래서 만들었어.
- 질문 10개 답변
- AI가 프로토타입 자동 생성
- 배포까지 자동

이제 "앱 아이디어 있어요" 하면
**3분 뒤에 링크**로 보여줄 수 있어.
궁금하면 댓글에 **"시스템"**. 스하리.

---

## 2) 요청 버튼 한 번 → 프로토타입

나는 만원코딩.
어제 진짜 미친 짓 했어.

**요청 버튼 1번 클릭** →
1. 관리자에게 알림
2. AI 에이전트 작업 시작
3. 프로토타입 코드 생성
4. Vercel 자동 배포
5. 결과 링크 전달

**엔드투엔드 자동화** 완성.
버튼 누르고 커피 한 잔 마시면 앱이 나와.

이거 가능하게 만든 게 "10고개 앱 빌더".
댓글에 **"자동화"** 남겨봐. 스하리.

## 3) 자동화 첫 단계: 버그/권한부터

나는 만원코딩.
자동화 멋있어 보이지?
근데 진짜 먼저 해야 할 건 따로 있어.

**버그 잡기 + 권한(RLS) 설정.**

이거 안 하고 자동화하면?
→ 사고 터짐. 데이터 새고, 권한 뚫림.

어제 하루 종일 한 일:
- RLS 정책 10개 수정
- API 라우트 권한 체크 추가
- Admin 전용 로직 분리

**화려한 건 나중. 기본기가 먼저.**
공감하면 댓글에 **"기본기"**. 스하리.

---

## 4) 결제 후 온보딩을 9단계로 만든 이유

나는 만원코딩.
결제 직후가 제일 위험해.
"됐다!" 하고 이탈하거든.

그래서 온보딩을 **9단계**로 쪼갰어.
1단계: "좋아요, 시작해볼까요?"
2단계: "어떤 느낌의 앱을 원하세요?"
...
9단계: "마지막 질문이에요!"

**핵심은 긴장감.**
끝이 보이니까 완주하게 되고,
완주하면 결과물이 나와.

이탈률 확 줄었어.
댓글에 **"9단계"** 남겨봐. 스하리.

## 5) 테스터 기능이 진짜 중요한 이유

나는 만원코딩.
서비스 런칭할 때 제일 무서운 거?
**피드백 없이 달리는 것.**

그래서 "테스터" 기능 만들었어.
- 관리자가 이메일로 테스터 지정
- 테스터는 무료로 전체 기능 사용
- 버그 리포트 바로 받음

**무료 → 피드백 → 개선 → 확산**
이 루프가 초기에 제일 중요해.

테스터 해보고 싶은 사람?
댓글에 **"테스터"**. 스하리.

## 6) 보안 없는 자동화는 사고 난다

나는 만원코딩.
AI 에이전트가 자동으로 코드 만들고 배포해?
멋지지. 근데 무서워야 해.

**보안 없으면 진짜 터져.**

그래서 3중 방어 만들었어:
1. 금지 키워드 필터링 (악성 입력 차단)
2. 이상 감지 시 즉시 중단
3. 모든 요청에 보안 지침 주입

자동화는 '빠름'이 아니라 '안전한 빠름'이어야 해.
공감하면 댓글에 **"보안"**. 스하리.

## 8) Vercel + Supabase에서 터지는 포인트

나는 만원코딩.
Next.js + Vercel + Supabase 조합?
최고인데, 자주 터지는 데가 있어.

**체크리스트:**
- [ ] RLS 켰는데 정책 안 넣음 → 무한 빈 배열
- [ ] Service Role Key를 클라이언트에서 씀 → 보안 헬
- [ ] Edge Function에서 DB 호출 → Cold Start 지옥
- [ ] 타입 불일치 → 런타임 에러

이거 하나씩 잡다 보면 하루 다 감.
공감하면 댓글에 **"체크"**. 스하리.

## 9) 프로토타입의 기준: 화면 5개면 충분

나는 만원코딩.
MVP 만들 때 "기능 더 넣어야지" 하다가
영원히 못 나오는 경우 많아.

**프로토타입은 딱 5개면 돼:**
1. 랜딩 (뭐하는 서비스?)
2. 로그인 (회원가입/로그인)
3. 목록 (메인 콘텐츠)
4. 상세 (개별 항목)
5. 마이페이지 (내 정보)

이 5개로 **서비스 흐름**이 다 보여.
나머지는 반응 보고 붙이면 돼.
댓글에 **"5개"**. 스하리.

나는 만원코딩.

- 질문 10개 → 프로토타입

개발은 '완성'이 아니라 '시작'이더라.
같이 지켜봐줘.
댓글에 **"운영"**. 스하리.
