# 🎙️ 돈 한 푼 안 쓰고 AI 음성 서버 만들었다 (Qwen3-TTS 구축 후기)

> "매달 TTS API 비용이 너무 아까운데... 그냥 직접 만들면 안 되나?"

이 한 마디에서 시작된 프로젝트입니다.

---

## 시작은 단순한 불만이었어요

유튜브 영상을 자동 생성하는 시스템을 만들다 보니 TTS(텍스트 음성 변환)가 필수였어요. ElevenLabs 같은 서비스를 쓰면 좋긴 한데... 영상 하나 만들 때마다 API 비용이 나가니까 규모가 커질수록 부담이 되더라고요.

그래서 생각했죠. **"집에 GPU 달린 서버 PC가 있잖아?"**

---

## GTX 1060으로 가능할까?

솔직히 처음엔 반신반의했어요. GTX 1060 6GB면 AI 작업용으로는 좀 빈약하거든요. 최신 AI 모델들은 보통 A100, 4090 이런 거 쓰잖아요.

근데 Qwen3-TTS 모델을 찾아보니까 **0.6B, 1.7B** 같은 경량 버전이 있더라고요. "이거라면 혹시?"

---

## 첫 번째 시도: 0.6B 모델 → 폭삭 망함 💀

```
CUDA assert input[0] != 0
```

이 오류 메시지 하나로 0.6B는 끝났습니다. 토크나이저 호환 문제라는데, 해결하려고 이것저것 해봤지만 결국 포기.

**1.7B-Base 모델로 갈아탔어요.**

그랬더니? **바로 됩니다.** 1060에서 음성이 술술 나왔어요. 진짜 이렇게 허무하게 해결될 줄이야.

---

## 근데 끝 글자가 자꾸 잘려

서버는 떴는데 문제가 생겼어요. 음성을 생성하면 마지막 1~2글자가 항상 잘리는 거예요.

"안녕하세요" → "안녕하세" 이런 식으로요.

처음엔 이렇게 해봤어요:
- 문장 길이 × 0.95 지점에서 자르기 → ❌ 속도마다 달라서 부정확
- "。" 문자를 패딩으로 넣기 → ❌ 토크나이저가 한자 패딩 못 씀

한참 삽질 끝에 나온 해법은 이거였어요:

**1. 텍스트 뒤에 더미 문장 붙이기**
```
"안녕하세요" + "...그리고 잠시 쉬어가겠습니다."
```

**2. 생성된 오디오에서 무음 구간 찾기**

**3. 무음 직전 지점에서 잘라내기**

생각해보면 당연한 거예요. 진짜 발화가 끝나면 무음이 오니까, 무음 감지가 가장 정확한 cut point인 거죠.

이 방법으로 **100% 깔끔하게 잘립니다.**

---

## 아키텍처: 역할 분리의 중요성

```
[내 PC] → API 요청 → [서버 PC 192.168.0.2:7860]
                           Qwen3-TTS가 음성 생성
                               ↓
                      C:\qwen-tts-server\output\
                      └── 날짜_프로젝트명\
                           └── 001.wav, 002.wav...
```

서버 PC는 **음성 생성만** 담당하고, 내 PC에서 **요청과 파일 관리**를 합니다. 이렇게 역할을 분리하니까 나중에 서버 바꿔도 코드 수정이 최소화돼요.

---

## 실제로 써봤더니

어제 유튜브 쇼츠 대본 23문장을 실제로 생성해봤어요. 총 90초 분량인데:

- **생성 시간**: 문장당 5~10초 내외
- **품질**: 아나운서 느낌보다는 자연스러운 편
- **끊김**: 무음 감지 방식으로 완벽하게 해결

API 비용? **0원.** 전기세만 좀 나오겠지만 그건 어쩔 수 없어요 ㅋㅋ

---

## 이게 왜 중요한가

지금은 유튜브 대본 TTS용이지만, 사실 이 서버 하나로 할 수 있는 게 엄청 많아요:

- 📹 유튜브/릴스 영상 나레이션 자동화
- 🎮 게임/앱 음성 출력
- 📚 텍스트 콘텐츠의 오디오북 변환
- 🤖 AI 챗봇 음성 응답

**인터넷 없어도 됩니다. API 키 없어도 됩니다. 24시간 무제한 사용 가능합니다.**

## 삽질 정리 (혹시 똑같이 고생하는 분을 위해)

| 오류 | 해결책 |
|------|--------|
| `CUDA assert input[0] != 0` | 0.6B 포기하고 1.7B 쓰세요 |
| HTTP 헤더에 한글 넣으면 에러 | 헤더에서 한글 제거 |
| `Torch not compiled with CUDA enabled` | venv 꼭 활성화하고 실행! |
| 끝 글자 잘림 | 더미 텍스트 + 무음 감지 방식 사용 |
