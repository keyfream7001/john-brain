## 포스팅 2 — 삽질 공유 (공감 유도)

```
오늘 AI 서버 만들면서 오류 3번 만남

1차: CUDA assert input[0] != 0
→ 0.6B 모델 문제. 1.7B로 교체

2차: latin-1 codec can't encode
→ HTTP 헤더에 한글 넣으면 안 됨

3차: Torch not compiled with CUDA enabled
→ venv 안 켜고 실행했던 것...

세 번 다 터지면서 배웠습니다 🙃
개발이 원래 이런 거지 뭐
```
