## 포스팅 3 — 꿀팁: Android 설치법

```
안드로이드에 OpenClaw 설치하는 법
(Termux 직접 설치는 안 됩니다)

Termux 기본 환경(Bionic libc)에서는
네이티브 Node 모듈 빌드가 안 됨

해결법:
proot-distro로 Ubuntu 환경 만들고
그 안에서 Node + npm + openclaw 설치

✅ proot-distro install ubuntu
✅ proot-distro login ubuntu
✅ npm install -g openclaw@latest

이러면 openclaw --version 2026.2.19-2 뜸
```
