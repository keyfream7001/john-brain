### [1단계: OpenClaw 업데이트 — 2:00~3:30]

**[화면: 터미널 업데이트 명령]**

먼저 OpenClaw 자체를 최신 버전으로 업데이트했어요.

```
openclaw update
```

버전이 `2026.2.6-3`에서 `2026.2.19-2`로 올라갔는데, 이번 업데이트에서 Claude Sonnet 4.6 지원이 추가됐습니다.

모델 설정 파일에 이걸 추가해주면 됩니다:
```json
"model": "anthropic/claude-sonnet-4-6"
```

맥북 노드도 LaunchAgent로 서비스 등록해서 재부팅해도 자동으로 연결되게 만들었어요.

### [2단계: AI 서버 PC 노드 구축 — 3:30~7:00]

**[화면: 서버 PC 화면 + 터미널]**

서버 PC는 GTX 1060 6GB짜리예요. 최신 GPU는 아니지만 AI 도구 돌리기엔 충분합니다.

**Tailscale 설치부터:**

```cmd
# 서버 PC에서
winget install Tailscale.Tailscale
# 로그인 후 같은 Tailnet 연결
```

**OpenClaw 노드 연결:**

```cmd
set OPENCLAW_GATEWAY_TOKEN=<your-token>
openclaw node run --host <gateway-host> --port 443 --tls --display-name "AI-Server"
```

**[화면: AI 도구 설치 과정 타임랩스]**

그 다음 AI 도구 6개 설치입니다.

```cmd
# Python 3.13 환경
pip install faster-whisper rembg torch ultralytics paddlepaddle paddleocr

# Python 3.11 venv (C 확장 패키지)
C:\ai-tools\venv311\Scripts\activate
pip install basicsr gfpgan realesrgan insightface onnxruntime-gpu
```

**⚠️ 주의 포인트!**

Real-ESRGAN이랑 InsightFace는 **반드시 Python 3.11 환경**에서 설치하세요. 3.13에서는 C 확장 빌드 오류가 납니다.

그리고 basicsr의 이 import 오류도 직접 수정해야 해요:

**[화면: 코드 에디터]**

```python
# 이걸
from torchvision.transforms.functional_tensor import rgb_to_grayscale
# 이걸로
from torchvision.transforms.functional import rgb_to_grayscale
```

**FastAPI 서버 구성:**

6개 엔드포인트를 가진 API 서버를 만들었어요.

```
/api/stt       → 음성 → 텍스트
/api/remove-bg → 배경 제거
/api/detect    → 객체 감지
/api/ocr       → 이미지 → 텍스트
/api/upscale   → 이미지 업스케일
/api/face-analyze → 얼굴 분석
```

### [3단계: 안드로이드 폰 연결 — 7:00~11:00]

**[화면: 안드로이드 폰 + Termux 터미널]**

이게 오늘 제일 어려웠던 파트예요. 솔직하게 보여드릴게요.

**첫 시도: Termux에서 직접 설치 → 실패**

```bash
npm install -g openclaw
# Error: koffi/spawn.h: No such file or directory
```

안드로이드의 Bionic libc가 GNU libc랑 달라서 네이티브 모듈 빌드가 안 됩니다.

**해결책: proot-distro Ubuntu**

```bash
# Termux에서
pkg install proot-distro
proot-distro install ubuntu
proot-distro login ubuntu

# Ubuntu 내부에서
apt update && apt upgrade -y
apt install -y curl git build-essential ca-certificates
curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
apt install -y nodejs

npm install -g openclaw@latest
openclaw --version  # 2026.2.19-2 ✅
```

**[화면: 노드 연결 성공 화면]**

그 다음 Tailscale 설치하고, openclaw node run 실행하면...

페어링 승인 화면이 뜨고, Approve 누르면 연결 완료!

**ADB로 자동화한 것들:**

```powershell
# Termux 설정 변경
adb shell run-as com.termux files/usr/bin/bash -c \
  "echo 'allow-external-apps=true' >> ~/.termux/termux.properties"
```
