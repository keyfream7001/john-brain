## 💰 가격 체계 (확정)

### 기본 원칙
> **3시간 내 작업 = 만원**
> 초과분은 시간당 5천원

| 구분 | 범위 | 가격 |
|------|------|------|
| 기본 | 3시간 내 | 만원 |
| 추가 | 3시간 초과 | 시간당 5천원 |
| 유지보수 | 월 관리 | 월 3~5만원 |
| 풀 패키지 | 기획~배포 | 별도 협의 |

### 만원으로 가능한 것 (3시간 내)
- 프로토타입 앱 (MVP)
- 웹사이트 10페이지 이내
- DB 스키마 설계
- API 연동 (여러 개 OK)
- 자동화 스크립트
- 크롤링/데이터 수집

### 전략: 락인 구조
```
만원에 확 해줌 → 감동 → 관리 못함 → 계속 의뢰 → 장기 고객
```
