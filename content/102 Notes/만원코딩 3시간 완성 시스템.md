# 만원코딩 3시간 완성 시스템

> 3시간 안에 프로젝트 완성하려면 이게 다 준비되어 있어야 함!

---

## 🎯 핵심 원칙

```
고객 준비 (1일 전) → 존 작업 (3시간) → 바로 배포
```

---

## 📋 1. 고객용 사전 가이드

### 🌐 웹사이트/랜딩페이지 요청 시

```markdown
# 🚀 프로젝트 시작 전 준비해주세요!

예상 소요시간: 10분

---

## Step 1. GitHub 가입 (코드 저장소)
1. https://github.com 접속
2. "Sign up" 클릭 → 이메일로 가입
3. 가입 완료 후 저에게 GitHub 아이디 알려주세요

---

## Step 2. 도메인 (인터넷 주소)

선택해주세요:
- [ ] 이미 있어요 → 주소 알려주세요
- [ ] 새로 사고 싶어요 → 원하는 이름 알려주세요 (예: myshop.com)
- [ ] 무료로 시작할래요 → OK!

---

## Step 3. 디자인 자료

있으면 보내주세요:
- [ ] 로고 이미지 (PNG, 고화질)
- [ ] 브랜드 색상 (예: #3B82F6)
- [ ] 참고하고 싶은 사이트 링크

---

## Step 4. 내용

- [ ] 회사/서비스 소개 텍스트
- [ ] 연락처 정보
- [ ] SNS 링크 (있으면)

---

✅ 다 준비되면 답장 주세요!
❓ 어려우면 화면공유로 도와드릴게요
```

### 🔐 로그인 기능 있는 서비스 요청 시

```markdown
# 🚀 프로젝트 시작 전 준비해주세요!

예상 소요시간: 20분

---

## Step 1. GitHub 가입
1. https://github.com 가입
2. GitHub 아이디 알려주세요

---

## Step 2. Supabase 가입 (데이터베이스)
1. https://supabase.com 접속
2. "Start your project" → GitHub로 로그인
3. 왼쪽 위 Organization 이름 알려주세요

---

## Step 3. 카카오 로그인 (선택)
1. https://developers.kakao.com 접속
2. 카카오 계정으로 로그인
3. "내 애플리케이션" → "애플리케이션 추가하기"
4. 앱 이름 아무거나 입력 → 저장
5. "앱 키" 페이지에서 REST API 키 복사해서 보내주세요

---

## Step 4. 구글 로그인 (선택)
1. https://console.cloud.google.com 접속
2. 새 프로젝트 만들기
3. 프로젝트 이름 알려주세요 (제가 나머지 세팅해드릴게요)

---

✅ 완료되면 정보 보내주세요!
```

## 🛠️ 2. 프로젝트 템플릿 (존용)

### 기본 스택
```
Next.js 14 + TypeScript + Tailwind CSS
├── Supabase (DB + Auth)
├── Vercel (배포)
└── shadcn/ui (컴포넌트)
```

### 템플릿 종류

| 템플릿 | 포함 기능 | 예상 시간 |
|--------|-----------|-----------|
| **landing** | 반응형 1페이지 | 1시간 |
| **website** | 다중 페이지 + 연락폼 | 2시간 |
| **auth** | 로그인/회원가입 | 2시간 |
| **dashboard** | 인증 + 대시보드 | 3시간 |
| **payment** | 인증 + 결제 | 3시간 |

### 템플릿 GitHub 레포
```
keyfream7001/1man-template-landing
keyfream7001/1man-template-auth
keyfream7001/1man-template-dashboard
```

### 새 프로젝트 시작 명령어
```bash
# 1. 템플릿 클론
gh repo create 고객프로젝트명 --template keyfream7001/1man-template-auth --private

# 2. 클론
git clone https://github.com/keyfream7001/고객프로젝트명

# 3. 의존성 설치
cd 고객프로젝트명 && npm install

# 4. 환경변수 복사
cp .env.example .env.local

# 5. 개발서버
npm run dev
```

## 📝 3. 환경변수 체크리스트

### .env.local 템플릿
```bash
# 사이트 기본
NEXT_PUBLIC_SITE_URL=https://example.com
NEXT_PUBLIC_SITE_NAME=사이트이름

# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx
SUPABASE_SERVICE_ROLE_KEY=eyJxxx

# 카카오 로그인
NEXT_PUBLIC_KAKAO_CLIENT_ID=xxx
KAKAO_CLIENT_SECRET=xxx

# 구글 로그인
GOOGLE_CLIENT_ID=xxx.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=xxx

# 결제 (키움페이)
KIWOOM_CPID=xxx
KIWOOM_AUTH_KEY=xxx
KIWOOM_WEBHOOK_SECRET=xxx
```
