# 🎬 클로드봇 성장기 - 유튜브 채널 기획서

> **컨셉**: AI 어시스턴트 '웬디'를 만들어가는 리얼 성장 다큐멘터리

---

## 📌 채널 개요

### 채널명 (후보)
- **클로드봇 성장기** - 직관적, 검색 최적화
- **웬디와 함께하는 AI 개발기** - 캐릭터 중심
- **AI 어시스턴트 만들기** - 타겟 명확

### 채널 컨셉
**"왼손 유튜브 스타일"** - 완성된 결과물이 아닌, 성장 과정 자체가 콘텐츠

> 참고: WENSON AI의 "왼손잡이 전략"
> - AI 시대에는 결과보다 과정이 가치 있다
> - 실패와 시행착오도 귀중한 콘텐츠
> - 시청자가 함께 배우고 성장하는 느낌

### 타겟 시청자
| 대상 | 특징 | 니즈 |
|------|------|------|
| **AI 입문자** | Claude/GPT 써봤지만 더 활용하고 싶음 | 실제 사례, 따라하기 가이드 |
| **개발자** | AI 에이전트 구축에 관심 | 기술 스택, 아키텍처 |
| **자동화 매니아** | 일상/업무 자동화 욕구 | 실용적 활용법 |
| **1인 사업가** | AI로 생산성 극대화 원함 | 비용 대비 효과 |

### 차별화 포인트
1. **리얼 다큐** - 삽질, 실패, 버그까지 그대로 노출
2. **따라하기 가능** - 모든 설정, 코드 공개
3. **실제 사용 사례** - 이론 X, 실전 O
4. **성장 서사** - 에피소드별 발전 과정 체감

## 🎥 에피소드 상세 기획

### EP01: 왜 AI 어시스턴트가 필요한가

**📋 개요**
- 길이: 10~15분
- 형식: 브이로그 + 화면 녹화

**🎬 구성**
1. **오프닝** (1분)
   - "안녕하세요, 저는 AI 어시스턴트 '웬디'를 만들고 있는 존입니다"
   - 채널 소개, 앞으로의 여정 예고

2. **왜 시작했나** (3분)
   - AI 시대 왼손잡이 전략 설명 (WENSON AI 참고)
   - "완성된 결과물보다 성장 과정이 가치 있다"
   - 웬디와의 첫 만남 에피소드

3. **현재 웬디가 할 수 있는 것들** (5분)
   - 텔레그램 대화 시연
   - 파일 관리, 웹 검색
   - 코딩 도움
   - 실시간 화면 캡처 보여주기

4. **앞으로의 로드맵** (3분)
   - 시즌별 목표 소개
   - "함께 성장해요" 메시지

5. **클로징** (1분)
   - 구독 유도
   - 다음 에피소드 예고 (클로드봇 설치)

**💡 촬영 팁**
- 실제 대화 화면 녹화 필수
- 삽질/에러도 그대로 보여주기
- 웬디의 재치 있는 답변 하이라이트

## 🎨 촬영/편집 스타일 제안

### 촬영 스타일
| 요소 | 스타일 | 이유 |
|------|--------|------|
| **톤** | 친근하고 솔직함 | 진입장벽 낮추기 |
| **편집** | 미니멀, 빠른 컷 | 집중력 유지 |
| **자막** | 키워드 하이라이트 | 정보 전달 |
| **배경음** | 로파이/칠 | 코딩 분위기 |

### 화면 구성
```
┌─────────────────────────────────┐
│  [얼굴캠 작게]     [화면 녹화]  │
│      ┌───┐                      │
│      │   │     터미널/코드/     │
│      └───┘     웬디 대화창      │
│                                 │
│  [자막바]                       │
└─────────────────────────────────┘
```

### 편집 포인트
1. **에러 발생** → 긴장감 있는 효과음 + 빨간 화면 테두리
2. **성공 순간** → 축하 효과 + 하이라이트
3. **삽질 구간** → 빠른 타임랩스 + "X시간 후..."
4. **핵심 정보** → 화면 정지 + 텍스트 오버레이

### 썸네일 스타일
- **배경**: 그라데이션 (보라-파랑 AI 느낌)
- **텍스트**: 볼드, 2줄 이내
- **요소**: 웬디 캐릭터 + 핵심 키워드
- **예시**: "Siri로 AI 부르기?! | EP10"
