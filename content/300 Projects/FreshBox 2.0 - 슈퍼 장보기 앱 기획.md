# 🧊 FreshBox 2.0 - 슈퍼 장보기 앱

> "마트에서 카메라만 켜면, 최저가 비교 + 자동 장바구니 + 냉장고 관리까지 한방에"

## 📍 기존 FreshBox
- 위치: `C:\JohnCodeQ\FreshBox\`
- 스택: Flutter + Supabase
- 상태: Phase 1 완료 (카메라 촬영 → AI 분석 → 식재료 등록)

---

## 🚀 확장 기능

### 1️⃣ 마트 실시간 장보기 도우미
- 제품 + 가격표 촬영 → 인터넷 최저가 즉시 비교
- 장바구니 실시간 합계 (카드 할인 반영)
- "이거 온라인이 3천원 더 싸요!" 알림

### 2️⃣ 오프라인 가격 크라우드소싱
- 사용자들이 찍은 가격 데이터 수집
- 이마트 vs 홈플러스 vs 코스트코 실시간 비교
- "우리 동네 어디가 제일 싼지" 지도

### 3️⃣ 원클릭 온라인 주문 연동
- "이건 쿠팡이 더 싸네" → 바로 주문
- 장보면서 온라인 최저가 동시 쇼핑
- 무거운 건 배달, 급한 건 직접 구매

### 4️⃣ 구매 → 냉장고 자동 등록
- 결제 완료 = FreshBox에 자동 입력
- 유통기한, 제품 정보 다 들어감
- 입력 귀찮음 완전 해결!

### 5️⃣ 글로벌 확장
- 나라별 온라인몰 크롤링
- 미국: Amazon, Walmart, Target
- 일본: Rakuten, Amazon JP
- 한국: 쿠팡, 마켓컬리, B마트

### 6️⃣ 노터치 카메라 UX
- 카메라 켜면 자동 인식 (셔터 X)
- 바코드/가격표 감지되면 바로 저장
- 손가락 하나 안 움직여도 됨

## 🎯 차별화 포인트

1. **입력 귀찮음 해결** - 카메라 자동 인식, 구매=자동등록
2. **오프라인+온라인 통합** - 마트에서 장보면서 온라인 최저가도 비교
3. **크라우드소싱 가격 DB** - 사용자가 많을수록 데이터 풍부
4. **글로벌 확장성** - 각 나라별 쇼핑몰 연동
