## 🏛️ 협회 게시판 시스템

### 신고 → 블랙리스트 흐름
```
매니저 신고접수 → 협회 관리자 검증
  → 접수 → 확인중 → 승인 → 블랙리스트 자동 추가
  → 또는 반려 (사유 기록)
대금 해소 시 → 협회 관리자가 해소/제거
```

### 페이지
- `/association` — 공지사항 (투표 분리)
- `/association/votes` — 투표 목록 + 실제 투표 + 관리자 생성
- `/association/reports` — 신고 접수 + 상태 추적
- `/association/blacklist` — 블랙리스트 목록 + 검색

### 관리자 대시보드
- admin 페이지에 3개 탭: 대시보드 / 🚨 신고 관리 / 🚫 블랙리스트 관리
